Disponibilità delle funzionalità

Non tutte le funzionalità di CloudFormation potrebbero essere disponibili in tutte le regioni. Per ulteriori informazioni sulle AWS regioni, consulta la tabella delle regioni dell'infrastruttura globale.

  • Le macros attualmente non sono disponibili nella seguente Regione:

    • Asia Pacifico (Giacarta)

  • Il rilevamento delle deviazioni al momento non è disponibile nella Regione:

    • Asia Pacifico (Osaka-Locale)

  • L'esecuzione di distribuzioni ECS blu/green tramite CodeDeploy l'utilizzo non AWS CloudFormationè attualmente disponibile nelle seguenti regioni:

    • Africa (Città del Capo)

    • Asia Pacifico (Osaka-Locale)

    • Europa (Milano)

StackSets e macro

StackSets attualmente non supporta la creazione o l'aggiornamento di set di stack con autorizzazioni gestite dal servizio da modelli che contengono macro. Ciò include le trasformazioni, che sono macro ospitate da. AWS CloudFormationPer ulteriori informazioni sulle macro, consulta Macro di modello.
