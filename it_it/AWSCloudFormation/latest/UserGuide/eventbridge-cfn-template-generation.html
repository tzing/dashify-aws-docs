Generazione CloudFormation di modelli per EventBridge bus, regole e pipe

Per aiutarti a iniziare subito a sviluppare CloudFormation modelli, la EventBridge console ti consente di creare CloudFormation modelli partendo dai bus di eventi, dalle regole e dalle pipe esistenti nel tuo account.

Per ulteriori dettagli, consulta i seguenti argomenti nella Amazon EventBridge User Guide:

  • Generazione CloudFormation di modelli da un bus di EventBridge eventi

  • Generazione CloudFormation di modelli da una EventBridge regola

  • Genera un AWS CloudFormation modello da EventBridge Pipes

Puoi anche creare direttamente modelli per bus e regole di eventi utilizzando AWS::Events::EventBus e altre EventBridge risorse. AWS::Events::Rule
