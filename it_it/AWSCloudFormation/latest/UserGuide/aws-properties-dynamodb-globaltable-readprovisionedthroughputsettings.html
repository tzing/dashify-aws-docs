AWS::DynamoDB::GlobalTable ReadProvisionedThroughputSettings

Consente di specificare le impostazioni della capacità di lettura per una tabella di replica o un indice secondario globale di replica quando BillingMode è impostata su PROVISIONED. Devi specificare un valore per ReadCapacityUnits o ReadCapacityAutoScalingSettings, ma non entrambi. Puoi passare dalla capacità fissa al dimensionamento automatico.

Sintassi

Per dichiarare questa entità nel modello AWS CloudFormation, utilizzare la sintassi seguente:

JSON

{
  "ReadCapacityAutoScalingSettings" : CapacityAutoScalingSettings,
  "ReadCapacityUnits" : Integer
}

YAML

  ReadCapacityAutoScalingSettings: 
    CapacityAutoScalingSettings
  ReadCapacityUnits: Integer

Proprietà

ReadCapacityAutoScalingSettings

Specifica le impostazioni di dimensionamento automatico per la tabella di replica o l'indice secondario globale.

Required: No

Tipo: CapacityAutoScalingSettings

Aggiornamento richiesto: nessuna interruzione

ReadCapacityUnits

Specifica una capacità di lettura fissa per la tabella di replica o per un indice secondario globale.

Required: No

Tipo: integer

Minimum: 1

Aggiornamento richiesto: nessuna interruzione
