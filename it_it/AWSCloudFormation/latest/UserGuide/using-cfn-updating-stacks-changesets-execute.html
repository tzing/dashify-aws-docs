Esecuzione di un set di modifiche

Per eseguire le modifiche descritte in un set di modifiche allo stack, esegui il set di modifiche.

Importante

Quando si esegue un set di modifiche, CloudFormation elimina tutti i set di modifiche aggiuntivi che sono associati allo stack perché non sono più validi per l'aggiornamento dello stack. Se l'aggiornamento ha esito negativo, devi creare un nuovo set di modifiche.

Policy stack ed esecuzione di un set di modifiche

Se si esegue un set di modifiche su uno stack che dispone di una policy stack associata, CloudFormation applica la policy quando aggiorna lo stack. Non è possibile specificare una policy di stack temporanea che sovrascrive la policy esistente quando esegui un set di modifiche. Per aggiornare una risorsa protetta, devi aggiornare la policy stack o utilizzare il metodo aggiornamento diretto.

Execute a change set for nested stacks (console)
Per eseguire un set di modifiche per gli stack nidificati (console)

  1. Nella console AWS CloudFormation, in Stacks (Stack), selezionare il nome dello stack che si desidera aggiornare. È necessario scegliere il nome dello stack associato al set di modifiche root.

  2. Nel riquadro di navigazione, selezionare Change Sets (Set di modifiche) per visualizzare un elenco del set di modifiche dello stack.

  3. Scegliere il nome del set di modifiche root che si desidera eseguire.

  4. Nella pagina dei dettagli del set di modifiche, scegliere Execute (Esegui).

    Nota

    CloudFormation esegue le modifiche descritte nel set di modifiche root e nei set di modifiche nidificati, se è stato selezionato Enabled (Abilitato) per i set di modifiche per gli stack nidificati durante il Creazione di un set di modifiche processo.

    CloudFormation avvia immediatamente l'aggiornamento dello stack. La console CloudFormation indirizza alla scheda Events (Eventi), dove è possibile monitorare l'avanzamento dell'aggiornamento stack.

Execute a change set (console)
Per eseguire un set di modifiche (console)

  1. Nella console AWS CloudFormation, in Stacks (Stack), selezionare il nome dello stack che si desidera aggiornare.

  2. Nel riquadro di navigazione, selezionare Change Sets (Set di modifiche) per visualizzare un elenco del set di modifiche dello stack.

  3. Selezionare il nome del set di modifiche che si desidera eseguire.

  4. Nella pagina dei dettagli del set di modifiche, scegliere Execute (Esegui).

    CloudFormation avvia immediatamente l'aggiornamento dello stack. La console CloudFormation indirizza alla scheda Events (Eventi), dove è possibile monitorare l'avanzamento dell'aggiornamento stack.

Per eseguire un set di modifiche (AWS CLI)

  • Esegui il comando aws cloudformation execute-change-set.

    Specificare l'ID set di modifiche che ha il set di modifiche che si desidera eseguire, come nell'esempio seguente:

    aws cloudformation execute-change-set --change-set-name arn:aws:cloudformation:us-east-1:123456789012:changeSet/SampleChangeSet/1a2345b6-0000-00a0-a123-00abc0abc000

    Il comando dell'esempio esegue un set di modifiche con l'ID arn:aws:cloudformation:us-east-1:123456789012:changeSet/SampleChangeSet/1a2345b6-0000-00a0-a123-00abc0abc000.

    Una volta eseguito il comando, CloudFormation avvia l'aggiornamento dello stack. Per visualizzare l'avanzamento dello stack, utilizzare il comando aws cloudformation describe-stacks.
