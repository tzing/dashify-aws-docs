Eliminazione di un set di modifiche

L'eliminazione di un set di modifiche lo rimuove dall'elenco dei set di modifiche per lo stack. L'eliminazione di un set di modifiche impedisce a un utente di eseguire accidentalmente un set di modifiche che non dovrebbe essere applicato. A meno che non vengano eliminati, CloudFormation conserva tutti i set di modifiche finché non si aggiorna lo stack.

Delete a change set for nested stacks
Per eliminare un set di modifiche per gli stack nidificati (console)

  1. Nella console AWS CloudFormation, in Stacks, selezionare il nome dello stack associato al set di modifiche root.

  2. Nel riquadro di navigazione, selezionare Change Sets (Set di modifiche) per visualizzare un elenco del set di modifiche dello stack.

  3. Selezionare il nome del set di modifiche che si desidera eliminare.

  4. Nella pagina dei dettagli del set di modifiche, scegliere Delete (Elimina). Scegliendo Delete (Elimina) si eliminerà l'intera gerarchia dei set di modifiche nidificati.

    Nota

    L'operazione di eliminazione per i set di modifiche per gli stack nidificati è asincrona e mostrerà uno stato DELETE_PENDING seguito da uno stato DELETE_IN_PROGRESS. Al termine dell'operazione di eliminazione del set di modifiche, i set di modifiche verranno rimossi dall'elenco. Anche gli stack nidificati nello stato REVIEW_IN_PROGRESS verranno eliminati se sono stati creati durante la creazione dei set di modifiche.

    CloudFormation inizia immediatamente a eliminare il set di modifiche dall'elenco dei set di modifiche dello stack.

    Nota

    Se disponi di stack nidificati bloccati in un'operazione in corso, consulta la sezione relativa alla risoluzione degli errori in Risoluzione dei problemi di AWS CloudFormation.

Delete a change set
Per eliminare un set di modifiche (console)

  1. Nella console AWS CloudFormation, in Stacks (Stack), selezionare il nome dello stack che contiene il set di modifiche che si desidera eliminare.

  2. Nel riquadro di navigazione, selezionare Change Sets (Set di modifiche) per visualizzare un elenco del set di modifiche dello stack.

  3. Selezionare il nome del set di modifiche che si desidera eliminare.

  4. Nella pagina dei dettagli del set di modifiche, scegliere Delete (Elimina).

    CloudFormation inizia immediatamente a eliminare il set di modifiche dall'elenco di set di modifiche dello stack e reindirizza l'utente alla pagina Stack.

Per eliminare un set di modifiche (AWS CLI)

  • Eseguire il comando aws cloudformation delete-change-set, specificando l'ID del set di modifiche che si desidera eliminare, come nell'esempio seguente:

    aws cloudformation delete-change-set --change-set-name arn:aws:cloudformation:us-east-1:123456789012:changeSet/SampleChangeSet/1a2345b6-0000-00a0-a123-00abc0abc000
