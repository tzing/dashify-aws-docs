Tipi di richieste di risorse personalizzate

Il tipo di richiesta viene inviato nel campo RequestType dell'oggetto richiesta del fornitore inviato da AWS CloudFormation quando il template developer crea, aggiorna o elimina uno stack che contiene una risorsa personalizzata.

Ogni tipo di richiesta possiede uno specifico set di campi che viene inviato con la richiesta, tra cui un URL Amazon S3 per la risposta del provider di risorse personalizzate. Il provider deve rispondere al bucket S3 con un risultato SUCCESS o FAILED entro un'ora. Dopo un'ora, la richiesta scade. Ogni risultato, inoltre, dispone di uno specifico set di campi previsto dal AWS CloudFormation.

Questa sezione fornisce informazioni sui campi di richiesta e risposta, con esempi per ogni tipo di richiesta.
