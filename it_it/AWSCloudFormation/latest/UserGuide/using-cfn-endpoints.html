Endpoint AWS CloudFormation

Per ridurre la latenza dei dati delle applicazioni, la maggior parte dei prodotti di Amazon Web Services consente di selezionare un endpoint regionale per le tue richieste. Un endpoint è un URL che rappresenta il punto di partenza per un servizio Web.

Quando utilizzi gli stack tramite l'interfaccia a riga di comando o le operazioni API, puoi specificare un endpoint regionale. Per ulteriori informazioni sulle regioni e sugli endpoint per AWS CloudFormation, consulta Regioni ed endpoint nella Riferimenti generali di Amazon Web Services.
