Distribuzione rapida dei modelli con trasformazioni

AWS CloudFormation richiede l'utilizzo di un set di modifiche per creare un modello che includa trasformazioni. Invece di creare e poi avviare in modo indipendente un set di modifiche, utilizza il comando aws cloudformation deploy. Quando viene eseguito, questo comando crea un set di modifiche, lo avvia, quindi viene terminato. Questo comando riduce il numero di fasi obbligatorie per la creazione o l'aggiornamento di uno stack che include trasformazioni.

Il comando seguente crea un nuovo stack tramite il modello my-template.json.

PROMPT> aws cloudformation deploy \
  --template /path_to_template/my-template.json \
  --stack-name my-new-stack \
  --parameter-overrides Key1=Value1 Key2=Value2 \

Per ulteriori informazioni, consulta il comando aws cloudformation deploy in Riferimento ai comandi AWS CLI.
