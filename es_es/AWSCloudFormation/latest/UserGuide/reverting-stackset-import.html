Reversión de una operación de importación de pilas

Para revertir una operación de importación de pila, lleve a cabo las siguientes tareas:

  1. Especifique un atributo DeletionPolicy de Retain para cada recurso que desee conservar después de eliminar la instancia de la pila. Para obtener más información, consulte Reversión de una operación de importación.

  2. Elimine las instancias de pila del conjunto de pilas. Para obtener más información, consulte Eliminación de instancias de pila del conjunto de pilas.

  3. Elimine el conjunto de pilas. Para obtener más información, consulte Eliminación de un conjunto de pilas.

Resultados: la operación de reversión eliminó la instancia de la pila y conservó los recursos de la pila.
