AWS::DynamoDB::GlobalTable ReplicaGlobalSecondaryIndexSpecification

Representa las propiedades de un índice secundario global que se puede configurar por réplica.

Sintaxis

Para declarar esta entidad en su plantilla de AWS CloudFormation, use la siguiente sintaxis:

JSON

{
  "ContributorInsightsSpecification" : ContributorInsightsSpecification,
  "IndexName" : String,
  "ReadProvisionedThroughputSettings" : ReadProvisionedThroughputSettings
}

YAML

  ContributorInsightsSpecification: 
    ContributorInsightsSpecification
  IndexName: String
  ReadProvisionedThroughputSettings: 
    ReadProvisionedThroughputSettings

Propiedades

ContributorInsightsSpecification

Actualiza el estado de la información de los colaboradores de una tabla o índice específico. CloudWatchLos gráficos de Contributor Insights for DynamoDB muestran la clave de partición y (si corresponde) la clave de clasificación de los elementos a los que se accede con frecuencia y los elementos restringidos con frecuencia en texto sin formato. Si necesita usar el Servicio de administración de AWS claves (KMS) para cifrar la clave de partición de esta tabla y ordenar los datos de la clave con una clave AWS administrada o una clave administrada por el cliente, no debe habilitar CloudWatch Contributor Insights for DynamoDB para esta tabla.

Obligatorio: no

Tipo: ContributorInsightsSpecification

Requisitos de actualización: sin interrupción

IndexName

El nombre del índice secundario global. Este nombre debe ser único entre todos los demás índices de esta tabla.

Obligatorio: sí

Tipo: cadena

Mínimo: 3

Máximo: 255

Requisitos de actualización: reemplazo

ReadProvisionedThroughputSettings

Le permite especificar la configuración de la capacidad de lectura de un índice secundario global de réplica cuando BillingMode se establece en PROVISIONED.

Obligatorio: no

Tipo: ReadProvisionedThroughputSettings

Requisitos de actualización: sin interrupción
