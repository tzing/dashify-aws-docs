Especificación del nombre de pila y los parámetros

Después de seleccionar una plantilla de pila, especifique el nombre de la pila y los valores de los parámetros que se definen en la plantilla.

Con los parámetros, puede personalizar la pila en el momento de su creación. Los valores de sus parámetros se pueden utilizar en la plantilla de pila para modificar la forma en que se configuran los recursos. De esta forma, no es necesario que codifique valores en varias plantillas para especificar diferentes ajustes. Para obtener más información sobre parámetros en una plantilla de AWS CloudFormation, consulte Parámetros.

Para especificar los valores del parámetro y el nombre de la pila

  1. En la página Specify stack (Especificar pila), escriba un nombre de pila en el cuadro Stack name (Nombre de pila).

    El nombre de pila es un identificador que le ayuda a encontrar una en particular en una lista de pilas. El nombre de una pila puede contener únicamente caracteres alfanuméricos (con distinción entre mayúsculas y minúsculas) y guiones. Debe comenzar con un carácter alfabético y no puede tener más de 128 caracteres.

  2. En la sección Parameters (Parámetros), especifique los parámetros que se definen en la plantilla de la pila.

    Puede utilizar o cambiar cualquier parámetro con valores predeterminados.

  3. Cuando esté satisfecho con los valores de los parámetros, elija Next (Siguiente) a fin de continuar con la configuración de opciones para la pila.

Tipos de parámetros específicos de AWS

Al crear pilas que contienen tipos de parámetros específicos de AWS, la consola de CloudFormation proporciona listas desplegables de valores válidos para esos parámetros. Dependiendo del tipo de parámetro, puede buscar valores por ID, nombre o el valor de la etiqueta Name. Por ejemplo, con el tipo de parámetro AWS::EC2::VPC::Id, puede buscar un ID de la VPC específico, como por ejemplo vpc-b47658d1. Si la VPC se etiquetó con un nombre, como por ejemplo Name:TestVPC, también puede buscar TestVPC. En la actualidad, puede buscar únicamente por valores de etiqueta con la clave Name.

nota

La consola no proporciona una lista desplegable ni le permite buscar valores con el tipo de parámetro AWS::EC2::Image::Id. CloudFormation solo verifica si los valores de entrada son ID válidos de imagen de Amazon Elastic Compute Cloud.

Agrupación y orden de parámetros

La consola ordena alfabéticamente parámetros por su ID lógico. Al crear una plantilla, puede utilizar la clave de metadatos AWS::CloudFormation::Interface para anular el orden predeterminado. Para obtener más información y un ejemplo de la clave de metadatos AWS::CloudFormation::Interface, consulte AWS::CloudFormation::Interface.
