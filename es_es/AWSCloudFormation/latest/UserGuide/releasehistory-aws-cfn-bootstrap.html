Historial de versiones de los scripts auxiliares de CloudFormation 1.4

La siguiente tabla proporciona acceso a versiones anteriores de los scripts auxiliares, que admiten Python versión 2.7 y 2.6. Se admite para compatibilidad con versiones anteriores. Recomendamos usar la versión más reciente de los scripts auxiliares, que requieren Python 3.4 o superior.

nota

Los scripts auxiliares de AWS CloudFormation están preinstalados en las imágenes de la AMI de Amazon Linux. Los paquetes de descarga que se muestran en la tabla se aplican a otras distribuciones de Linux/Unix y Microsoft Windows (2008 o posterior). Para obtener información sobre cómo utilizar los scripts auxiliares, consulte Referencia de scripts auxiliares de CloudFormation.
