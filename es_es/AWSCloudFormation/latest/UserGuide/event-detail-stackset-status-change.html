Detalle del evento de cambio de estado de StackSet

A continuación, se muestran los campos de detalle de los eventos de cambio de estado de StackSet.

Los campos source y detail-type se incluyen porque contienen valores específicos para los eventos de AWS CloudFormation.

{
  . . .,
  "detail-type": "CloudFormation StackSet Status Change",
  "source": "aws.cloudformation",
  . . .,
  "detail": {
    "stack-set-arn" : "string",
    "status-details": {
        "status":"string"
    }
  }
}
detail-type

Identifica el tipo de evento.

En el caso de los eventos de eventos de estado de StackSet, este valor es CloudFormation StackSet Status Change.

source

Identifica el servicio que generó el evento. En el caso de los eventos de CloudFormation, este valor es aws.cloudformation.

detail

Un objeto JSON que contiene información sobre el evento. El servicio que genera el evento determina el contenido de este campo.

En el caso de los eventos de evento de estado de StackSet, estos datos incluyen:

stack-set-arn

El nombre de recurso de Amazon (ARN) asociado al conjunto de la pila.

status-details
status

El estado de StackSet.

Valores válidos: ACTIVE | DELETED
