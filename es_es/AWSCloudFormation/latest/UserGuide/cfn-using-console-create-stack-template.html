Selección de una plantilla de pila

Después de iniciar el asistente de creación de pila, debe especificar la plantilla que desea que AWS CloudFormation utilice para la creación de la pila.

Las plantillas de CloudFormation son archivos de formato JSON o YAML que especifican los recursos de AWS que componen la pila. Para obtener más información acerca de las plantillas de CloudFormation, consulte Trabajo con plantillas de AWS CloudFormation.

Para elegir una plantilla de pila:

  1. En la página Specify Template (Especificar plantilla), elija una plantilla de pila utilizando una de las siguientes opciones:

    Template is ready (La plantilla está lista)

    Especifique una plantilla completada que está lista para crear una pila.

    En la sección Specify template (Especificar plantilla), seleccione la opción correspondiente en función de la ubicación de la plantilla:

    • URL de Amazon S3

      Especifique una URL para una plantilla en un bucket de S3.

      Escriba la URL en el campo Amazon S3 URL (URL de Amazon S3).

      importante

      Si la plantilla incluye pilas anidadas (por ejemplo, pilas descritas en otros documentos de plantilla ubicados en subdirectorios), asegúrese de que su bucket de S3 contiene los archivos y directorios necesarios.

      Si tiene una plantilla en un bucket compatible con el control de versiones, puede especificar una versión concreta de la plantilla, como https://s3.amazonaws.com/templates/myTemplate.template?versionId=123ab1cdeKdOW5IH4GAcYbEngcpTJTDW. Para obtener más información, consulte Managing objects in a versioning-enabled bucket (Administración de objetos en un bucket con control de versiones habilitado) en la Guía del usuario de la consola de Amazon Simple Storage Service.

      La URL debe dirigir a una plantilla con un tamaño máximo de 1 MB que esté almacenada en un bucket de S3 en el que tengas permisos de lectura. La URL puede tener una longitud máxima de 1024 caracteres. Algunos recursos pueden requerir que el bucket esté en la misma región que la pila.

    • Upload a template file (Cargar un archivo de plantilla)

      Seleccione una plantilla de CloudFormation de su equipo local.

      Elija Choose File (Elegir archivo) para seleccionar el archivo de plantilla que desea cargar. La plantilla puede tener un tamaño máximo de 1 MB. Una vez que haya elegido la plantilla, CloudFormation carga el archivo y muestra la URL de S3.

      Si usa la AWS CLI o la API para crear una pila, puede cargar una plantilla con un tamaño máximo de 51 200 bytes.

      nota

      Si carga un archivo de plantilla local, CloudFormation lo carga a un bucket de Amazon Simple Storage Service (Amazon S3) en su cuenta de AWS. Si todavía no tiene un bucket de S3 creado con CloudFormation, crea un bucket único para cada región en la que carga un archivo de plantilla. Si ya tiene un bucket de S3 creado con AWS CloudFormation en su cuenta de AWS, CloudFormation agregará la plantilla a dicho bucket.

      Consideraciones que deben tenerse en cuenta acerca de los buckets de S3 creados con CloudFormation

      • Cualquiera con permisos de Amazon S3 en su cuenta de AWS puede acceder a los buckets.

      • CloudFormation crea los buckets con el cifrado del lado del servidor habilitado de forma predeterminada; por lo tanto, cifra todos los objetos almacenados en el bucket.

        Puede administrar directamente las opciones de cifrado para los buckets creados con CloudFormation; por ejemplo, mediante la consola de Amazon S3 en https://console.aws.amazon.com/s3/ o la AWS CLI. Para obtener más información, consulte Amazon S3 default encryption for S3 buckets (Cifrado predeterminado de Amazon S3 para buckets de S3) en la Guía del usuario de Amazon Simple Storage Service.

      • Puede utilizar su propio bucket y administrar sus permisos cargando manualmente las plantillas a Amazon S3. Al crear o actualizar una pila, especifique la URL de Amazon S3 de un archivo de plantilla.

    Use a sample template (Usar una plantilla de muestra)

    • Para ver más fragmentos y muestras de plantillas, organizados por servicio de AWS, haga clic en View more sample templates (Ver más plantillas de muestra).

    Create template in Designer (Crear plantilla en Designer)

    Cree o modifique una plantilla con AWS CloudFormation Designer, una interfaz para arrastrar y colocar gráficamente diagramas para las plantillas. Para obtener más información, consulte ¿Qué es AWS CloudFormation Designer?.

  2. Para aceptar los ajustes, elija Next (Siguiente) y continúe con la especificación del nombre de la pila y los parámetros.

    Antes de crear recursos, CloudFormation valida su plantilla para identificar errores de sintaxis y algunos de semántica, como las dependencias circulares. Durante la validación, CloudFormation comprueba primero si la plantilla es un JSON válido. Si no lo es, CloudFormation comprueba si la plantilla es un YAML válido. Si ambas comprobaciones fallan, CloudFormation devuelve un error de validación de plantilla.
