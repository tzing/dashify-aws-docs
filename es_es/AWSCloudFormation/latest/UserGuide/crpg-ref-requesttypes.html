Tipos de solicitud de recurso personalizados

El tipo de solicitud se envía en el campo RequestType en el objeto de solicitud del proveedor que AWS CloudFormation envía cuando el desarrollador de plantillas crea, actualiza o elimina una pila que contiene un recurso personalizado.

Cada tipo de solicitud tiene un conjunto de campos determinado que se envía con la solicitud, incluida una URL de Amazon S3 para la respuesta del proveedor de recursos personalizados. El proveedor debe responder al bucket de S3 con un resultado de SUCCESS o FAILED en el plazo de una hora. Después de una hora, se agota el tiempo de espera de la solicitud. Cada resultado también tiene un conjunto determinado de campos previstos por AWS CloudFormation.

Esta sección proporciona información acerca de la solicitud y los campos de respuesta, con ejemplos, para cada tipo de solicitud.
