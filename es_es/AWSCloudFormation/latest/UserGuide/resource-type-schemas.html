Esquemas de proveedores de recursos de CloudFormation

Además de la especificación de recursos, CloudFormation pone a disposición los esquemas de proveedores de recursos para los recursos de Amazon por región. Cada esquema de proveedor de recursos es un archivo de texto con formato JSON que define las propiedades y atributos de un recurso específico.

Para cada región, CloudFormation ofrece un archivo ZIP que contiene los archivos de esquema individuales para cada recurso disponible en esa región. Para descargar los esquemas del proveedor de recursos para una región específica, consulte la tabla siguiente. También puede ver el esquema de proveedor de un recurso mediante el registro de CloudFormation.

La disponibilidad de recursos puede variar según la región. Para comprobar la disponibilidad de un recurso en una región determinada, consulte los esquemas de proveedor de recursos de esa región.

Para obtener más información acerca de los esquemas de proveedores de recursos, incluidos el formato y las definiciones del esquema, consulte Esquema del proveedor de recursos en la Guía del usuario de la interfaz de línea de comandos de CloudFormation.
