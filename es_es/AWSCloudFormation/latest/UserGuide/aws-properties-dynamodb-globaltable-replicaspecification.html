AWS::DynamoDB::GlobalTable ReplicaSpecification

Define la configuración específica de una única réplica de una tabla global.

Sintaxis

Para declarar esta entidad en su plantilla de AWS CloudFormation, use la siguiente sintaxis:

JSON

{
  "ContributorInsightsSpecification" : ContributorInsightsSpecification,
  "DeletionProtectionEnabled" : Boolean,
  "GlobalSecondaryIndexes" : [ ReplicaGlobalSecondaryIndexSpecification, ... ],
  "KinesisStreamSpecification" : KinesisStreamSpecification,
  "PointInTimeRecoverySpecification" : PointInTimeRecoverySpecification,
  "ReadProvisionedThroughputSettings" : ReadProvisionedThroughputSettings,
  "Region" : String,
  "SSESpecification" : ReplicaSSESpecification,
  "TableClass" : String,
  "Tags" : [ Tag, ... ]
}

YAML

  ContributorInsightsSpecification: 
    ContributorInsightsSpecification
  DeletionProtectionEnabled: Boolean
  GlobalSecondaryIndexes: 
    - ReplicaGlobalSecondaryIndexSpecification
  KinesisStreamSpecification: 
    KinesisStreamSpecification
  PointInTimeRecoverySpecification: 
    PointInTimeRecoverySpecification
  ReadProvisionedThroughputSettings: 
    ReadProvisionedThroughputSettings
  Region: String
  SSESpecification: 
    ReplicaSSESpecification

Propiedades

ContributorInsightsSpecification

La configuración utilizada para habilitar o deshabilitar CloudWatch Contributor Insights para la réplica especificada. Si no se especifica, el valor predeterminado es la información del colaborador deshabilitada para la réplica.

Obligatorio: no

Tipo: ContributorInsightsSpecification

Requisitos de actualización: sin interrupción

DeletionProtectionEnabled

Determina si una réplica está protegida contra la eliminación. Cuando está habilitada, ningún usuario o proceso puede eliminar la tabla. Este ajuste está deshabilitado de forma predeterminada. Para obtener más información, consulte Uso de la protección contra eliminaciones en la Guía para desarrolladores de Amazon DynamoDB.

Obligatorio: no

Tipo: booleano

Requisitos de actualización: sin interrupción

GlobalSecondaryIndexes

Define configuraciones adicionales para los índices secundarios globales de esta réplica.

Obligatorio: no

Tipo: matriz de ReplicaGlobalSecondaryIndexSpecification

Requisitos de actualización: sin interrupción

KinesisStreamSpecification

Define la configuración de Kinesis Data Streams para la réplica especificada.

Obligatorio: no

Tipo: KinesisStreamSpecification

Requisitos de actualización: sin interrupción

PointInTimeRecoverySpecification

La configuración utilizada para habilitar la recuperación a un momento dado. Si no se especifica, el valor predeterminado es la recuperación a un punto en el tiempo deshabilitada para la réplica.

Obligatorio: no

Tipo: PointInTimeRecoverySpecification

Requisitos de actualización: sin interrupción

ReadProvisionedThroughputSettings

Define la configuración de capacidad de lectura de la tabla de réplica.

Obligatorio: no

Tipo: ReadProvisionedThroughputSettings

Requisitos de actualización: sin interrupción

Region

La región en la que existe esta réplica.

Obligatorio: sí

Tipo: cadena

Requisitos de actualización: sin interrupción

SSESpecification

Permite especificar una clave administrada por el cliente para la réplica. Cuando se utilizan claves administradas por el cliente para el cifrado del lado del servidor, esta propiedad debe tener un valor en todas las réplicas.

Obligatorio: no

Tipo: ReplicaSSESpecification

Requisitos de actualización: sin interrupción

TableClass

La clase de tabla de la tabla especificada. Los valores válidos son STANDARD y STANDARD_INFREQUENT_ACCESS.

Obligatorio: no

Tipo: cadena

Valores permitidos: STANDARD | STANDARD_INFREQUENT_ACCESS

Requisitos de actualización: sin interrupción

Tags

Una matriz de pares clave-valor que aplicar a esta réplica.

Para obtener más información, consulte la sección sobre etiquetas.

Obligatorio: no

Tipo: matriz de etiquetas

Requisitos de actualización: sin interrupción
