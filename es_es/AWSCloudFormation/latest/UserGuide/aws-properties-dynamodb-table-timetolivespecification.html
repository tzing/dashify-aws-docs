AWS::DynamoDB::Table TimeToLiveSpecification

Representa la configuración utilizada para habilitar o deshabilitar el tiempo de vida (TTL) para la tabla especificada.

Sintaxis

Para declarar esta entidad en su plantilla de AWS CloudFormation, use la siguiente sintaxis:

JSON

{
  "AttributeName" : String,
  "Enabled" : Boolean
}

YAML

  AttributeName: String
  Enabled: Boolean

Propiedades

AttributeName

El nombre del atributo de TTL utilizado para almacenar la hora de vencimiento de los elementos de la tabla.

nota

  • La AttributeName propiedad es obligatoria cuando se habilita el TTL o cuando el TTL ya está activado.

  • Para actualizar esta propiedad, primero debe deshabilitar el TTL y, a continuación, habilitar el TTL con el nuevo nombre de atributo.

Obligatorio: condicional

Tipo: cadena

Mínimo: 1

Máximo: 255

Requisitos de actualización: sin interrupción

Enabled

Indica si TTL se habilita (true) o deshabilita (false) en la tabla.

Obligatorio: sí

Tipo: booleano

Requisitos de actualización: sin interrupción
