Eliminación de un conjunto de cambios

Al eliminar un conjunto de cambios se elimina de la lista de conjuntos de cambio de la pila. Eliminar un conjunto de cambios impide que usted u otro usuario ejecuten accidentalmente un conjunto de cambios que no debería aplicarse. CloudFormation conserva todos los conjuntos de cambios hasta que actualice la pila, salvo que los elimine.

Delete a change set for nested stacks
Eliminar un conjunto de cambios para pilas anidadas (consola)

  1. En la consola de AWS CloudFormation, en Stacks (Pilas), seleccione el nombre de pila asociado al conjunto de cambios raíz.

  2. En el panel de navegación, elija Change Sets (Conjuntos de cambios) para ver una lista con los conjuntos de cambios de la pila.

  3. Seleccione el nombre del conjunto de cambios que desea eliminar.

  4. En la página de detalles del conjunto de cambios, seleccione Delete (Eliminar). Al seleccionar Delete (Eliminar), eliminará toda la jerarquía de conjuntos de cambios anidados.

    nota

    La operación de eliminación de conjuntos de cambios para pilas anidadas es asíncrona y mostrará un estado DELETE_PENDING seguido de un estado DELETE_IN_PROGRESS. Una vez finalizada la operación de eliminación del conjunto de cambios, los conjuntos de cambios se eliminarán de la lista. Las pilas anidadas en el estado REVIEW_IN_PROGRESS también se eliminarán si se crearon durante la creación del conjunto de cambios.

    CloudFormation comienza inmediatamente a eliminar el conjunto de cambios de la lista de conjuntos de cambios de la pila.

    nota

    Si dispone de pilas anidadas atascadas en una operación en curso, consulte Solución de problemas en Solución de problemas de AWS CloudFormation.

Delete a change set
Para eliminar un conjunto de cambios (consola)

  1. En la consola de AWS CloudFormation, en Stacks (Pilas), seleccione el nombre de la pila que contenga el conjunto de cambios que desea eliminar.

  2. En el panel de navegación, elija Change Sets (Conjuntos de cambios) para ver una lista con los conjuntos de cambios de la pila.

  3. Seleccione el nombre del conjunto de cambios que desea eliminar.

  4. En la página de detalles del conjunto de cambios, seleccione Delete (Eliminar).

    CloudFormation comienza inmediatamente a eliminar el conjunto de cambios de la lista de conjuntos de cambios de la pila y lo redirige a la página Stacks (Pilas).

Eliminar un conjunto de cambios (AWS CLI)

  • Ejecute el comando aws cloudformation delete-change-set, especificando el ID del conjunto de cambios que desea eliminar, tal y como se muestra en el ejemplo siguiente:

    aws cloudformation delete-change-set --change-set-name arn:aws:cloudformation:us-east-1:123456789012:changeSet/SampleChangeSet/1a2345b6-0000-00a0-a123-00abc0abc000
