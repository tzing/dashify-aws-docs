Explicaciones

Las plantillas son archivos de texto con formato JSON o YAML que describen los recursos de AWS que desea aprovisionar o actualizar en sus pilas de AWS CloudFormation. Para crear plantillas, puede utilizar AWS CloudFormation Designer o un editor de texto.

Los siguientes procedimientos le ayudan a crear plantillas de AWS CloudFormation utilizando AWS CloudFormation Designer y texto sin formato.

  • Tutorial: Utilice AWS CloudFormation Designer para modificar una plantilla de la pila
  • Tutorial: Interconectar con una VPC en otra Cuenta de AWS
  • Tutorial: Consulte las salidas de recursos en otra pila de AWS CloudFormation
  • Tutorial: Creación de una aplicación con escalado y equilibrio de carga
  • Implementación de aplicaciones en Amazon EC2 con AWS CloudFormation
  • Creación de condiciones de espera en una plantilla
