Listado de recursos

Inmediatamente después de ejecutar el comando aws cloudformation create-stack, puede enumerar sus recursos mediante el comando aws cloudformation list-stack-resources. Este comando enumera un resumen de cada recurso en la pila especificado con el parámetro --stack-name. El informe incluye un resumen de la pila, incluido el estado de creación o eliminación.

El siguiente ejemplo muestra los recursos para la pila myteststack:

PROMPT> aws cloudformation list-stack-resources --stack-name myteststack
{
    "StackResourceSummaries": [
        {
            "ResourceStatus": "CREATE_COMPLETE",
            "ResourceType": "AWS::S3::Bucket",
            "ResourceStatusReason": null,
            "LastUpdatedTimestamp": "2013-08-23T01:02:28.025Z",
            "PhysicalResourceId": "myteststack-s3bucket-sample",
            "LogicalResourceId": "S3Bucket"
        }
    ]
}

AWS CloudFormation informa sobre los detalles de los recursos en cualquier pila en ejecución o eliminada. Si especifica el nombre de una pila cuyo estado es CREATE_IN_PROCESS, AWS CloudFormation informa solamente sobre aquellos recursos cuyo estado es CREATE_COMPLETE.

nota

El comando aws cloudformation describe-stack-resources devuelve información sobre pilas eliminadas hace 90 días una vez que se hayan eliminado.
