Registro de procedimientos almacenados

Los detalles sobre los procedimientos almacenados se registran en las siguientes vistas y tablas del sistema:

  • SVL_STORED_PROC_CALL: se registran detalles acerca de la hora de inicio y finalización de la llamada al procedimiento almacenado y acerca de si la llamada finaliza antes de la conclusión. Para obtener más información, consulte SVL_STORED_PROC_CALL.

  • SVL_STORED_PROC_MESSAGES: los mensajes de los procedimientos almacenados emitidos por la consulta RAISE se registran con el nivel de registro correspondiente. Para obtener más información, consulte SVL_STORED_PROC_MESSAGES.

  • SVL_QLOG: se registra el ID de consulta de la llamada al procedimiento para cada consulta llamada desde un procedimiento almacenado. Para obtener más información, consulte SVL_QLOG.

  • STL_UTILITYTEXT: se registran llamadas al procedimiento almacenado cuando están completadas. Para obtener más información, consulte STL_UTILITYTEXT.

  • PG_PROC_INFO: esta vista de catálogo del sistema muestra información acerca de procedimientos almacenados. Para obtener más información, consulte PG_PROC_INFO.
