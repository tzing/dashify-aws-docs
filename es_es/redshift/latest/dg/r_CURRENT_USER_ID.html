CURRENT_USER_ID

Devuelve un identificador único para el usuario de Amazon Redshift que ha abierto la sesión actual.

Sintaxis

CURRENT_USER_ID

Tipo de retorno

La función CURRENT_USER_ID devuelve un valor entero.

Ejemplos

El siguiente ejemplo devuelve el nombre de usuario y el ID de usuario actual para esta sesión: 

select user, current_user_id;

 current_user | current_user_id
--------------+-----------------
   dwuser     |               1
(1 row)
