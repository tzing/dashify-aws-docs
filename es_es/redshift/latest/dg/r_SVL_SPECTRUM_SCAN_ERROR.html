SVL_SPECTRUM_SCAN_ERROR

Puede consultar la vista del sistema SVL_SPECTRUM_SCAN_ERROR para obtener información acerca de los errores de análisis de Redshift Spectrum.

SVL_SPECTRUM_SCAN_ERROR es visible para todos los usuarios. Los superusuarios pueden ver todas las filas; los usuarios normales solo pueden ver sus datos. Para obtener más información, consulte Visibilidad de datos en las tablas y vistas de sistema.

Algunos o todos los datos de esta tabla también están en la vista de monitoreo SYS SYS_EXTERNAL_QUERY_ERROR. Los datos de la vista de monitoreo SYS están formateados para que sean más fáciles de usar y entender. Se recomienda utilizar la vista de monitoreo SYS para las consultas.

Columnas de la tabla

Muestra un ejemplo de errores registrados. El valor predeterminado es 10 entradas por consulta.

Nombre de la columna Tipo de datos Descripción
userid integer ID del usuario que generó la consulta.
consulta integer ID de la consulta que generó esta fila.
location character (128) Ubicación de los datos que se están consultando.
rowid character (128) La ubicación del error en el archivo. Las partes de rowid se separan con : (dos puntos), y es posible agregar otras partes en el futuro. 
row_offset:row_group:row_id
Un row_offset es el desplazamiento (en bytes) de la fila dentro del archivo, y se establece en -1 para formatos de archivo no admitidos. Una tabla se divide en row_groups, y cada grupo tiene filas con distintos row_ids.
colname character (128) Nombre de la columna que ha devuelto la consulta.
original_value character (128) Valor original consultado.
modified_value character (128) Valor modificado devuelto en función de la opción de configuración de control de datos especificada en la consulta.
Desencadenador character (128) Opción de control de datos especificada en la consulta.
acción character (128) Acción asociada a la opción de control de datos especificada en la consulta.
action_value character (128) Valor del parámetro de acción asociado a la opción de control de datos especificada en la consulta.
error_code integer Código del resultado de la opción de control de datos especificada en la consulta.
