Función ABS

ABS calcula el valor absoluto de un número, donde ese número puede ser un valor literal o una expresión que tome el valor de un número.

Sintaxis

ABS(number)

Argumentos

número

Número o expresión que toma el valor de un número. Puede ser el tipo SMALLINT, INTEGER, BIGINT, DECIMAL, FLOAT4, FLOAT8 o SUPER.

Tipo de retorno

ABS devuelve el mismo tipo de datos como su argumento.

Ejemplos

Para calcular el valor absoluto de -38, utilice el ejemplo siguiente. 

SELECT ABS(-38);

+-----+
| abs |
+-----+
|  38 |
+-----+

Para calcular el valor absoluto de (14-76), utilice el ejemplo siguiente. 

SELECT ABS(14-76);

+-----+
| abs |
+-----+
|  62 |
+-----+
