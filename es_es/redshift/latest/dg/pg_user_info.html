Columnas de la tabla

PG_USER_INFO

PG_USER_INFO es una vista del sistema de Amazon Redshift que muestra información del usuario, como el ID de usuario y el tiempo de caducidad de la contraseña.

Solo los superusuarios pueden ver PG_USER_INFO.

Columnas de la tabla

PG_USER_INFO contiene las siguientes columnas. Para obtener más información, consulte la documentación de PostgreSQL.

Nombre de la columna Tipo de datos Descripción
usename nombre El nombre de usuario.
usesysid entero El ID de usuario.
usecreatedb booleano True si el usuario puede crear bases de datos.
usesuper booleano True si el usuario es un superusuario.
usecatupd booleano True si el usuario puede actualizar los catálogos del sistema.
passwd texto La contraseña.
valuntil abstime Fecha y hora de caducidad de la contraseña.
useconfig text[] El valor predeterminado de la sesión para las variables en tiempo de ejecución.
useconnlimit texto Número de conexiones que el usuario puede abrir.
