ST_M

ST_M devuelve la coordenada m de un punto de entrada.

Sintaxis

ST_M(point)

Argumentos

point

Un valor POINT de tipo de datos GEOMETRY.

Tipo de retorno

Valor DOUBLE PRECISION de la coordenada m.

Si point es null, entonces se devuelve el valor null.

Si point es un punto 2D o 3DZ, se devuelve el valor null.

Si point es un punto vacío, se devuelve el valor null.

Si point no es POINT, se devuelve un error.

Ejemplos

El siguiente SQL devuelve la coordenada m de un punto en una geometría 3DM. 

SELECT ST_M(ST_GeomFromEWKT('POINT M (1 2 3)'));

st_m
-----------
 3

El siguiente SQL devuelve la coordenada m de un punto en una geometría 4D. 

SELECT ST_M(ST_GeomFromEWKT('POINT ZM (1 2 3 4)'));

st_m
-----------
 4
