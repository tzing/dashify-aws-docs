Memuat kolom tipe data VARBYTE

Anda dapat memuat data dari file dalam format CSV, Parket, dan ORC. Untuk CSV, data dimuat dari file dalam representasi heksadesimal dari data VARBYTE. Anda tidak dapat memuat data VARBYTE dengan FIXEDWIDTH opsi. REMOVEQUOTESOpsi ADDQUOTES atau COPY tidak didukung. Kolom VARBYTE tidak dapat digunakan sebagai kolom partisi.
