CloudFormation skema penyedia sumber daya

Selain spesifikasi sumber daya, menyediakan CloudFormation skema penyedia sumber daya untuk sumber daya Amazon menurut Wilayah. Setiap skema penyedia sumber daya adalah file teks berformat JSON yang menentukan properti dan atribut dari sumber daya tertentu.

Untuk setiap Wilayah, CloudFormation menawarkan file zip yang berisi file skema individual untuk setiap sumber daya yang tersedia di Wilayah tersebut. Untuk mengunduh skema penyedia sumber daya untuk Wilayah tertentu, lihat tabel berikut. Anda juga dapat melihat skema penyedia sumber daya menggunakan Registri CloudFormation .

Ketersediaan sumber daya dapat bervariasi menurut Wilayah. Untuk memeriksa ketersediaan sumber daya di Wilayah tertentu, lihat skema penyedia sumber daya untuk Wilayah tersebut.

Untuk mempelajari selengkapnya tentang skema penyedia sumber daya, termasuk format dan definisi skema, lihat Skema penyedia sumber daya di Panduan Pengguna Command Line Interface CloudFormation .
