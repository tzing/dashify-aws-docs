Menghapus set perubahan

Menghapus set perubahan menghapusnya dari daftar set perubahan untuk tumpukan. Menghapus set perubahan mencegah Anda atau pengguna lain secara tidak sengaja menjalankan set perubahan yang seharusnya tidak diterapkan. Kecuali Anda menghapusnya, CloudFormation mempertahankan semua set perubahan sampai Anda memperbarui tumpukan.

Delete a change set for nested stacks
Untuk menghapus set perubahan untuk tumpukan nest (konsol)

  1. Di konsol AWS CloudFormation, di Tumpukan, pilih nama tumpukan yang berhubungan dengan set perubahan root.

  2. Di panel navigasi, pilih Set Perubahan untuk melihat daftar set perubahan tumpukan.

  3. Pilih nama set perubahan yang ingin Anda hapus.

  4. Pada halaman detail set perubahan, pilih Hapus. Dengan memilih Hapus, Anda akan menghapus seluruh hirarki set perubahan nest.

    catatan

    Operasi hapus untuk set perubahan untuk tumpukan nest adalah asinkron dan akan menunjukkan status DELETE_PENDING, diikuti oleh status DELETE_IN_PROGRESS. Setelah menyelesaikan operasi hapus set perubahan, set perubahan akan dihapus dari daftar. Tumpukan nest di status REVIEW_IN_PROGRESS juga akan dihapus jika dibuat selama pembuatan set perubahan.

    CloudFormation mulai menghapus set perubahan dengan segera dari daftar tumpukan set perubahan.

    catatan

    Jika Anda memiliki tumpukan bersarang yang macet dalam operasi yang sedang berlangsung, lihat Pemecahan Masalah Kesalahan di Pemecahan Masalah AWS CloudFormation.

Delete a change set
Untuk menghapus set perubahan (konsol)

  1. Di konsol AWS CloudFormation, di Tumpukan, pilih nama tumpukan yang berisi set perubahan yang ingin Anda hapus.

  2. Di panel navigasi, pilih Set Perubahan untuk melihat daftar set perubahan tumpukan.

  3. Pilih nama set perubahan yang ingin Anda hapus.

  4. Pada halaman detail set perubahan, pilih Hapus.

    CloudFormation mulai menghapus set perubahan dengan segera dari daftar tumpukan set perubahan, dan Anda diarahkan ke halaman Tumpukan.

Untuk menghapus set perubahan (AWS CLI)

  • Jalankan perintah aws cloudformation delete-change-set, menentukan ID set perubahan yang ingin Anda hapus, seperti yang ditunjukkan dalam contoh berikut:

    aws cloudformation delete-change-set --change-set-name arn:aws:cloudformation:us-east-1:123456789012:changeSet/SampleChangeSet/1a2345b6-0000-00a0-a123-00abc0abc000
