Penentuan nama tumpukan dan parameter

Setelah memilih templat tumpukan, tentukan nama tumpukan dan nilai untuk parameter yang ditentukan di templat.

Dengan parameter, Anda dapat mengkustomisasi tumpukan pada saat pembuatan. Nilai parameter Anda dapat digunakan di templat tumpukan untuk mengubah cara sumber daya dikonfigurasi. Dengan begitu Anda tidak perlu melakukan hard-coding pada nilai di beberapa templat untuk menentukan berbagai pengaturan. Untuk informasi lebih lanjut tentang parameter di templat AWS CloudFormation, lihat Parameter.

Untuk menentukan nama tumpukan dan nilai parameter

  1. Pada halaman Tentukan detail tumpukan, ketik nama tumpukan di kotak Nama tumpukan.

    Nama tumpukan adalah pengidentifikasi yang membantu Anda menemukan tumpukan tertentu dari daftar tumpukan. Sebuah nama tumpukan dapat berisi karakter alfanumerik (peka huruf besar/kecil) dan tanda hubung. Ini harus dimulai dengan karakter abjad dan tidak boleh lebih dari 128 karakter.

  2. Di bagian Parameter, tentukan parameter yang ditentukan di templat tumpukan.

    Anda dapat menggunakan atau mengubah parameter dengan nilai default.

  3. Bila Anda puas dengan nilai parameter, pilih Selanjutnya untuk melanjutkan dengan pilihan pengaturan untuk tumpukan Anda.

AWS-jenis parameter tertentu

Saat Anda membuat tumpukan yang berisiAWS-jenis parameter tertentu, CloudFormation konsol menyediakan daftar tarik-turun dari nilai yang valid untuk parameter tersebut. Tergantung dari jenis parameter, Anda dapat mencari nilai menurut ID, nama, atau nilai tanda Name. Misalnya, dengan jenis parameter AWS::EC2::VPC::Id, Anda dapat mencari ID VPC tertentu, seperti vpc-b47658d1. Jika VPC ditandai dengan nama, seperti Name:TestVPC, Anda juga dapat mencari TestVPC. Saat ini, Anda hanya dapat mencari nilai tanda dengan kunci Name.

catatan

Konsol tidak menyediakan daftar drop-down atau memungkinkan Anda untuk mencari nilai denganAWS::EC2::Image::Idjenis parameter. CloudFormation hanya memverifikasi apakah nilai input adalah ID image Amazon Elastic Compute Cloud yang valid.

Mengelompokkan dan mengurutkan parameter

Konsol secara abjad mencantumkan parameter input menurut ID logis mereka. Ketika membuat templat, Anda dapat menggunakan kunci metadata AWS::CloudFormation::Interface untuk menimpa pengurutan default. Untuk informasi lebih lanjut dan contoh dari kunci metadata AWS::CloudFormation::Interface, lihat AWS::CloudFormation::Interface.
