Meninjau tumpukan Anda di konsol AWS CloudFormation

Langkah terakhir sebelum tumpukan Anda diluncurkan adalah meninjau nilai yang dimasukkan saat membuat tumpukan.

  1. Pada halaman Peninjauan, tinjau detail tumpukan Anda.

    Jika Anda perlu mengubah salah satu nilai sebelum peluncuran tumpukan, pilih Edit pada bagian yang sesuai untuk kembali ke halaman yang memiliki pengaturan yang ingin Anda ubah.

  2. Setelah Anda meninjau pengaturan pembuatan tumpukan, pilih Buat tumpukan untuk meluncurkan tumpukan Anda.

    catatan

    Mulai titik ini, Anda juga dapat memilih untuk membuat set perubahan baru daripada tumpukan baru. Untuk melakukannya, klik Buat set perubahan daripada Buat tumpukan. Untuk informasi selengkapnya, lihat Pembuatan tumpukan menggunakan set perubahan

    CloudFormation menampilkan panel Acara dari halaman Rincian tumpukan untuk tumpukan baru Anda. Dari sini, Anda dapat melihat peristiwa, data, atau sumber daya tumpukan Anda. CloudFormation secara otomatis menyegarkan peristiwa tumpukan setiap menit. Selain itu, CloudFormation menampilkan lencana Acara baru tersedia saat acara tumpukan baru terjadi; pilih ikon segarkan untuk memuat acara ini ke dalam daftar. Dengan melihat acara pembuatan tumpukan, Anda dapat memahami urutan acara yang mengarah ke pembuatan tumpukan Anda (atau kegagalan, jika Anda mengawakutu tumpukan Anda).

    Sembari tumpukan Anda dibuat, ia dicantumkan pada halaman Tumpukan dengan status CREATE_IN_PROGRESS.

    Setelah tumpukan Anda berhasil dibuat, statusnya berubah menjadi CREATE_COMPLETE. Anda kemudian dapat memilih tab Output untuk melihat output tumpukan jika Anda telah menentukannya di templat.

Menentukan akar penyebab kegagalan

Jika tumpukan Anda gagal dibuat, AWS CloudFormation dapat membantu Anda menentukan peristiwa yang kemungkinan merupakan akar penyebab kegagalan tumpukan. Untuk melakukan ini, dalam daftar tumpukan pilih tumpukan gagal dan kemudian pilih Deteksi akar penyebab.

CloudFormation akan menganalisis kegagalan dan menunjukkan peristiwa yang kemungkinan merupakan akar penyebab kegagalan.
