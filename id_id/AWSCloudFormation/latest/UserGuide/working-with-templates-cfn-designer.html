ApaAWS CloudFormationDesainer?

AWS CloudFormationDesainer (Designer) adalah alat grafis untuk membuat, dan mengubahAWS CloudFormationtemplat. Dengan Designer, Anda dapat diagram sumber daya template Anda menggunakan drag-and-dropantarmuka, dan kemudian edit detailnya menggunakan editor JSON dan YAKL terintegrasi. Apakah Anda seorang yang baru atau berpengalamanAWS CloudFormationpengguna,AWS CloudFormationDesainer dapat membantu Anda dengan cepat melihat keterkaitan antara sumber daya templat dan mengubah templat dengan mudah.

Desainer adalah bagian dariAWS CloudFormationkonsol Untuk menggunakannya, buka Designer dihttps://console.aws.amazon.com/cloudformation/designerdan masuk denganAWSkredensi
