AWS CloudFormation StackSets contoh template

Bagian ini mencakup tautan ke beberapa sampel AWS CloudFormation templat yang dapat membantu Anda menggunakan AWS CloudFormation StackSets di korporasi Anda. Templat yang tercantum di bagian ini mengaktifkan AWS CloudTrail atau AWS Config dan aturan di dalamnya.

penting

Sebagai praktik keamanan terbaik saat mengizinkan AWS Config akses ke bucket Amazon S3, kami sangat menyarankan Anda membatasi akses dalam kebijakan bucket dengan kondisi tersebut. AWS:SourceAccount Template baru diperbarui untuk memilikiAWS:SourceAccount. Jika kebijakan bucket Anda yang ada tidak mengikuti praktik terbaik keamanan ini, kami sangat menyarankan Anda mengedit kebijakan bucket tersebut untuk menyertakan perlindungan ini. Hal ini memastikan bahwa AWS Config diberikan akses atas nama pengguna yang diharapkan saja.

Aktifkan AWS Config dengan pendataan pusat https://s3.amazonaws.com/ cloudformation-stackset-sample-templates AWSConfigForOrganizations -US-east-1/Aktifkan .yml.yl

Konfigurasi sebuah AWS Config aturan untuk menentukan apakah EIP terpasang https://s3.amazonaws.com/ cloudformation-stackset-sample-templates -us-east-1/ ConfigRuleEipAttached .yl
