Pengonfigurasian gerbang akun target di AWS CloudFormation StackSets

Gerbang akun adalah fitur opsional yang memungkinkan Anda menentukan fungsi AWS Lambda untuk memverifikasi bahwa akun target memenuhi persyaratan tertentu sebelum AWS CloudFormation StackSets memulai operasi tumpukan di akun itu. Contoh umum gerbang akun adalah memverifikasi bahwa tidak ada CloudWatch alarm yang aktif atau belum terselesaikan di akun target. StackSets memanggil fungsi setiap kali Anda memulai operasi tumpukan di akun target, dan hanya berlanjut jika fungsi menampilkanSUCCEEDED kode. Jika fungsi Lambda menampilkan statusFAILED, StackSets tidak diteruskan dengan operasi yang Anda minta. Jika Anda tidak mengonfigurasi fungsi Lambda gerbang akun, StackSets melompati pemeriksaan, dan meneruskan dengan operasi Anda.

Jika akun target Anda menggagalkan pemeriksaan gerbang akun, operasi yang digagalkan akan dihitung terhadap jumlah toleransi kegagalan yang ditentukan atau persentase tumpukan. Untuk informasi lebih lanjut tentang toleransi kegagalan, lihat Opsi operasi set tumpukan.

Gerbang akun hanya tersedia untuk StackSets operasi. Fungsionalitas ini tidak tersedia untukAWS CloudFormation operasi lain di luar StackSets.

Persyaratan penyiapan

Daftar berikut menjelaskan persyaratan penyiapan untuk gerbang akun.

  • Untuk menggunakan fungsionalitas gerbang StackSets akun, fungsi Lambda Anda harus diberi nama AWSCloudFormationStackSetAccountGate.

  • Izin AWSCloudFormationStackSetExecutionRolemembutuhkan untuk memanggil fungsi Lambda Anda. Tanpa izin ini, StackSets melompati pemeriksaan gerbang akun, dan meneruskan dengan operasi tumpukan.

  • InvokeFunctionIzin Lambda harus ditambahkan ke akun target agar gerbang akun berfungsi. Kebijakan kepercayaan akun target harus memiliki hubungan kepercayaan dengan akun administrator. Berikut ini adalah contoh pernyataan kebijakan yang memberikaninvokefunction izin Lambda.

    {
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Action": [
                "lambda:InvokeFunction"
            ],
            "Resource": "*"
        }
    ]
}
