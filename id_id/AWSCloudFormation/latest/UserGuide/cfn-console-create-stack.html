Membuat tumpukan pada konsol AWS CloudFormation

Sebelum Anda membuat tumpukan, Anda harus memiliki templat yang menggambarkan sumber daya AWS CloudFormation apa yang akan termasuk dalam tumpukan Anda. Untuk informasi selengkapnya, lihat Bekerja dengan templat AWS CloudFormation.

catatan

Untuk melakukan pratinjau tumpukan baru, Anda dapat menggunakan set perubahan.

Membuat tumpukan pada CloudFormation konsol adalah proses mudah, yang didorong panduan yang terdiri dari langkah-langkah berikut:

  1. Memulai panduan Buat Tumpukan

  2. Memilih templat tumpukan

  3. Menentukan parameter tumpukan

  4. Mengatur opsi tumpukan AWS CloudFormation

  5. Meninjau tumpukan Anda

Setelah membuat tumpukan, Anda dapat memantau kemajuan tumpukan, melihat sumber daya dan output tumpukan, memperbarui tumpukan, dan menghapusnya. Informasi tentang tindakan ini disediakan dalam topik yang terkait.

Memulai panduan Buat Tumpukan

Untuk membuat tumpukan pada CloudFormation konsol

  1. Buka AWS CloudFormation konsol di https://console.aws.amazon.com/cloudformation.

  2. Membuat tumpukan baru dengan menggunakan salah satu opsi berikut:

    • Pilih Buat tumpukan. Ini adalah satu-satunya opsi jika Anda memiliki tumpukan yang sedang berjalan.

    • Pilih Buat Tumpukan, pada halaman Tumpukan. Opsi ini hanya terlihat jika Anda tidak memiliki tumpukan yang sedang berjalan.

Selanjutnya, Anda pilih templat tumpukan.
