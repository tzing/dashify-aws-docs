Hapus set tumpukan

Ketika Anda selesai dengan panduan Memulai AWS CloudFormation StackSets , Anda dapat mengikuti prosedur di bagian ini untuk menghapus set tumpukan dan sumber daya lain yang telah Anda buat sebagai bagian dari panduan ini. Untuk menghapus set tumpukan, Anda harus terlebih dahulu menghapus semua instans tumpukan di set tumpukan. Untuk informasi tentang cara menghapus semua instans tumpukan, lihat Hapus instans tumpukan dari set tumpukan Anda.

Hapus set tumpukan menggunakan AWS Management Console

  1. Pada halaman StackSets, pilih set tumpukan yang Anda buat di Buat set tumpukan. Dalam panduan ini, kita membuat set tumpukan bernama my-awsconfig-stackset.

  2. Dengan set tumpukan yang dipilih, pilih Hapus StackSet dari menu Tindakan.

  3. Ketika Anda diminta untuk mengonfirmasi bahwa Anda ingin menghapus kumpulan tumpukan, pilih Hapus StackSet.

Hapus set tumpukan menggunakan AWS CLI

Ketika bertindak sebagai administrator yang didelegasikan, Anda harus mengatur parameter --call-as ke DELEGATED_ADMIN setiap kali Anda menjalankan perintah StackSets.

--call-as DELEGATED_ADMIN

  1. Jalankan perintah berikut. Saat Anda diminta untuk mengonfirmasi, ketik y, lalu tekan Masuk.

    aws cloudformation delete-stack-set --stack-set-name my-awsconfig-stackset

  2. Verifikasi bahwa set tumpukan telah dihapus dengan menjalankan perintah list-stack-sets. Hasil list-stack-sets perintah harus menunjukkan tumpukan Anda dengan statusDELETED.

    aws cloudformation list-stack-sets

Hapus peran layanan (opsional)

Hapus peran layanan yang StackSets diperlukan untuk pembuatan set tumpukan.

Untuk set tumpukan yang dikelola sendiri, peran yang Anda buat sebagai bagian dari Prasyarat untuk operasi set tumpukan untuk panduan dalam panduan ini diberi nama AWSCloudFormationStackSetAdministrationRole di akun administrator, dan AwsCloudFormationStackSetExecutionRole di setiap akun target.

Untuk kumpulan tumpukan yang dikelola layanan, peran yang dibuat secara otomatis StackSets sebagai bagian dari panduan dalam panduan ini memiliki akhiran di akun manajemen organisasi, dan CloudFormationStackSetsOrgAdmin CloudFormationStackSetsOrgMember di setiap akun target. Prasyarat untuk operasi set tumpukan

Untuk informasi selengkapnya tentang menghapus peran, lihat Menghapus peran dan profil instance di Panduan Pengguna IAM.

Untuk menghapus peran layanan dengan menggunakan AWS Management Console

  1. Masuk ke AWS Management Console dan buka konsol IAM di https://console.aws.amazon.com/iam/.

  2. Di panel navigasi, pilih Peran, lalu isi kotak centang di samping peran yang ingin Anda hapus.

  3. Untuk menu Tindakan peran di bagian atas halaman, pilih Hapus peran.

  4. Di kotak dialog konfirmasi, pilih Ya, Hapus. Jika Anda yakin, Anda dapat melanjutkan penghapusan meskipun data terakhir mengakses layanan masih memuat.

Untuk menghapus peran layanan dengan menggunakan AWS CLI

  • Jalankan perintah berikut. Saat Anda diminta untuk mengonfirmasi, ketik y, lalu tekan Masuk.

    aws iam delete-role --role-name role name
