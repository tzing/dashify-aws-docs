Fn::Transform

Fungsi intrinsik Fn::Transform menentukan makro untuk melakukan pemrosesan kustom pada bagian dari templat tumpukan. Macro memungkinkan Anda untuk melakukan pemrosesan kustom pada template, dari tindakan sederhana sepertifind-and-replaceoperasi untuk transformasi ekstensif dari seluruh template. Untuk informasi selengkapnya, lihat Menggunakan AWS CloudFormation makro untuk melakukan pemrosesan khusus pada templat.

Anda juga dapat menggunakan Fn::Transform untuk memanggil transform Tansformasi AWS::Include, yang merupakan makro yang di-host oleh AWS CloudFormation.

Pernyataan

JSON

Sintaks untuk nama fungsi penuh:

{
    "Fn::Transform": {
        "Name": "macro name",
        "Parameters": {
            "Key": "value"
        }
    }
}

Sintaks untuk bentuk pendek:

{
    "Transform": {
        "Name": "macro name",
        "Parameters": {
            "Key": "value"
        }
    }
}

YAML

Sintaks untuk nama fungsi penuh:

Fn::Transform:
  Name : macro name
  Parameters :
    Key : value

Sintaks untuk bentuk pendek:

!Transform
  Name: macro name
  Parameters:
    Key: value

Parameter

Nama

Nama makro yang ingin Anda lakukan pemrosesan.

Parameter

Daftar parameter, ditetapkan sebagai pasangan kunci-nilai, untuk dilewatkan ke makro.

Nilai kembali

Cuplikan templat yang diproses untuk dimasukkan dalam templat tumpukan yang diproses.

Contoh

Contoh berikut memanggilAWS::Includetransform, menentukan bahwa lokasi untuk mengambil cuplikan template dari dilewatkan diInputValueparameter.

JSON

{
    "Fn::Transform": {
        "Name": "AWS::Include",
        "Parameters": {
            "Location": {
                "Ref": "InputValue"
            }
        }
    }
}

YAML

'Fn::Transform':
  Name: 'AWS::Include'
  Parameters:
    Location: !Ref InputValue

Contoh berikut memanggilAWS::Includetransform, menentukan bahwa lokasi untuk mengambil cuplikan template dari terletak diRegionMappemetaan, di bawah kuncinyaus-east-1dan kunci bersarangs3Location.

JSON

{
    "Fn::Transform": {
        "Name": "AWS::Include",
        "Parameters": {
            "Location": {
                "Fn::FindInMap": [
                    "RegionMap",
                    "us-east-1",
                    "s3Location"
                ]
            }
        }
    }
}

YAML

!Transform
  Name: AWS::Include
  Parameters:
    Location: !FindInMap
      - RegionMap
      - us-east-1
      - s3Location

Fungsi yang didukung

Tidak ada.

CloudFormationmelewati setiap panggilan fungsi intrinsik termasuk dalamFn::Transformke makro yang ditentukan sebagai string literal. Untuk informasi selengkapnya, lihat AWS CloudFormation antarmuka fungsi makro.
