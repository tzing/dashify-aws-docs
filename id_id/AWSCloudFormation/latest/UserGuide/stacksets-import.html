Mengimpor tumpukan ke AWS CloudFormation StackSets

Operasi impor AWS CloudFormation tumpukan dapat mengimpor tumpukan yang ada ke set tumpukan baru atau yang sudah ada, sehingga Anda dapat memigrasi tumpukan yang ada ke tumpukan yang ditetapkan dalam satu operasi. StackSetsmemperluas fungsionalitas tumpukan, sehingga Anda dapat membuat, memperbarui, atau menghapus tumpukan di beberapa akun dan Wilayah dengan satu operasi.

Gunakan operasi impor tumpukan untuk dikelola sendiri atau dikelola layananStackSets. Untuk dikelola sendiriStackSets, operasi impor dapat mengimpor tumpukan di akun administrator atau di akun target yang berbeda dan. Wilayah AWS Untuk dikelola layananStackSets, operasi impor dapat mengimpor tumpukan apa pun yang AWS Organizations sama dengan akun manajemen. Operasi impor dapat mengimpor hingga 10 tumpukan menggunakan ID tumpukan inline atau hingga 200 tumpukan menggunakan objek. Amazon S3

Impor tumpukan tersedia di mana pun StackSets didukung. Untuk informasi tentang dukungan StackSets Wilayah, lihat dukungan StackSets regional.

Persyaratan untuk impor tumpukan

Karena stack set melakukan operasi tumpukan di beberapa akun, sebelum Anda dapat membuat set tumpukan pertama, Anda memerlukan izin yang diperlukan yang ditentukan di akun AndaAkun AWS.

  • Untuk menyiapkan izin yang diperlukan untuk membuat set tumpukan dengan izin yang dikelola sendiri, lihat Melakukan operasi set tumpukan melibatkan wilayah yang dinonaktifkan secara default dan Memberikan izin yang dikelola sendiri.

  • Untuk menyiapkan izin yang diperlukan untuk membuat set tumpukan dengan izin yang dikelola layanan, lihat Melakukan operasi set tumpukan melibatkan wilayah yang dinonaktifkan secara default dan Aktifkan akses tepercaya dengan AWS Organizations.

Sebelum Anda mengimpor tumpukan ke dalam kumpulan tumpukan, pastikan Anda memahami persyaratan berikut:

  • Tumpukan hanya bisa menjadi milik satu set tumpukan.

  • Anda dapat menerapkan tag tumpukan ke tumpukan yang ditetapkan dengan menentukan tag secara eksplisit sebagai parameter dalam operasi impor tumpukan.

  • Penggantian parameter kustom tumpukan tidak terpengaruh selama operasi impor.

  • Contoh StackSets kuota dan tumpukan berlaku saat mengimpor tumpukan. Untuk informasi lebih lanjut tentang kuota, lihat AWS CloudFormationkuota.
