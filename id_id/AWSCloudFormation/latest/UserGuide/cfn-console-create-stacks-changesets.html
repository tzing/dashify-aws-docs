Pembuatan tumpukan menggunakan set perubahan

Untuk mem-pratinjau cara tumpukan AWS CloudFormation akan dikonfigurasi sebelum pembuatan tumpukan, membuat set perubahan. Fungsi ini memungkinkan Anda untuk memeriksa beragam konfigurasi dan membuat koreksi dan perubahan pada tumpukan Anda sebelum menjalankan set perubahan. Untuk informasi lebih lanjut tentang set perubahan, lihat Memperbarui tumpukan menggunakan set perubahan.

Pembuatan set perubahan untuk stack baru

Untuk membuat set perubahan untuk tumpukan baru, pilih templat tumpukan Anda dan tentukan konfigurasi tumpukan seperti yang Anda lakukan jika membuat tumpukan baru, lalu pilih untuk membuat set perubahan baru daripada tumpukan baru.

Untuk membuat set perubahan menggunakan CloudFormation konsol

  1. Memulai Buat wizard tumpukan

  2. Memilih templat tumpukan

  3. Menentukan parameter untuk tumpukan Anda

  4. Mengatur pilihan tumpukan

  5. Pada halaman Peninjauan, tinjau detail tumpukan Anda.

    Jika Anda perlu mengubah salah satu pengaturan sebelum membuat set perubahan, pilih Edit pada bagian yang sesuai untuk kembali ke halaman yang memiliki pengaturan yang ingin Anda ubah.

  6. Pilih Buat set perubahan.

  7. Masukkan nama untuk set perubahan, dan deskripsinya jika mau. Pilih Buat set perubahan.

    Ketika Anda membuat set perubahan untuk tumpukan baru, CloudFormation melakukan hal berikut:

    • Meluncurkan tumpukan baru dengan status REVIEW_IN_PROGRESS.

    • Membuat set perubahan untuk tumpukan baru yang mencerminkan konfigurasi tumpukan yang Anda tentukan di langkah sebelumnya.

    CloudFormation menampilkanSet perubahanhalaman untuk tumpukan yang diusulkan. Sembari CloudFormation membuat set perubahan, statusnya adalah CREATE_IN_PROGRESS, dan status eksekusinya adalah UNAVAILABLE. Saat CloudFormation selesai berhasil menciptakan set perubahan, ia menetapkan status perubahan set untukCREATE_COMPLETE, dan status pelaksanaannya adalahAVAILABLE. Status tumpukan diperbarui ke REVIEW_IN_PROGRESS. Pada titik ini, Anda dapat menjalankan set perubahan untuk menyelesaikan pembuatan tumpukan baru.

    Di panel Perubahan, CloudFormation menampilkan konfigurasi yang diajukan untuk tumpukan Anda.

    Jika CloudFormation gagal membuat set perubahan, ia mengatur status set perubahan keCREATE_FAILED. Perbaiki kesalahan yang ditampilkan di bidang Alasan status, lalu buat set perubahan baru. Pada tahap ini, Anda dapat mencoba berbagai konfigurasi dan membuat koreksi dan perubahan pada tumpukan Anda sebelum menjalankan set perubahan selanjutnya.

  8. Untuk menyelesaikan pembuatan tumpukan baru berdasarkan set perubahan, pilihEKSEKUSI, tentukan konfigurasi rollback Anda, lalu pilihJalankan set perubahan.
