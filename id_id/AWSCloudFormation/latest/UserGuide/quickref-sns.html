Snippet templat SNS Amazon

Contoh ini menampilkan sumber daya topik Amazon SNS. Ini membutuhkan alamat email yang valid.

JSON

"MySNSTopic" : {
    "Type" : "AWS::SNS::Topic",
    "Properties" : {
        "Subscription" : [ {
            "Endpoint" : "add valid email address",
            "Protocol" : "email"
        } ]
    }
}

YAML

MySNSTopic:
  Type: AWS::SNS::Topic
  Properties:
    Subscription:
    - Endpoint: "add valid email address"
      Protocol: email
