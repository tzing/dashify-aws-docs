Bekerja dengan templat AWS CloudFormation

Untuk menyediakan dan mengonfigurasi sumber daya tumpukan Anda, Anda harus memahami templat AWS CloudFormation, yang merupakan file teks yang diformat dalam JSON atau YAML. Templat ini menjelaskan sumber daya yang ingin Anda sediakan di tumpukan AWS CloudFormation. Anda dapat menggunakan AWS CloudFormation Designer atau editor teks apa pun untuk membuat dan menyimpan templat. Untuk informasi selengkapnya tentang struktur dan sintaks dari templat, lihat Anatomi templat.

Jika Anda tidak terbiasa dengan JSON atau YAMAL, Anda dapat menggunakan AWS CloudFormation Designer untuk membantu Anda memulai dengan template. AWS CloudFormation AWS CloudFormation Desainer adalah alat untuk membuat dan memodifikasi template secara visual. Untuk informasi selengkapnya, lihat ApaAWS CloudFormationDesainer?.

Cuplikan templat menyediakan contoh yang menunjukkan cara menulis templat untuk sumber daya tertentu. Misalnya, Anda dapat melihat cuplikan untuk instans Amazon EC2, domain Amazon S3, pemetaan, dan lainnyaAWS CloudFormation. Cuplikan dikelompokkan berdasarkan sumber daya, dengan tujuan umum cuplikan AWS CloudFormation di Snippet templat umum.

Untuk detail tentang sumber daya yang didukung, nama jenis, fungsi intrinsik, dan parameter semu yang dapat Anda gunakan di templat, lihat Referensi templat.
