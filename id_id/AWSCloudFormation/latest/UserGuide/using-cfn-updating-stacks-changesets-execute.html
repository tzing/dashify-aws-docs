Menjalankan set perubahan

Untuk membuat perubahan yang dijelaskan dalam set perubahan ke tumpukan Anda, jalankan set perubahan.

penting

Setelah Anda menjalankan set perubahan, CloudFormation menghapus setiap set perubahan tambahan yang berhubungan dengan tumpukan karena tidak lagi valid untuk tumpukan yang diperbarui. Jika pembaruan gagal, Anda perlu membuat set perubahan baru.

Kebijakan Tumpukan dan Menjalankan Set Perubahan

Jika Anda menjalankan set perubahan pada tumpukan yang memiliki kebijakan tumpukan yang berhubungan dengannya, CloudFormationmemberlakukan kebijakan ketika memperbarui tumpukan. Anda tidak dapat menentukan kebijakan tumpukan sementara yang menimpa kebijakan yang ada saat Anda menjalankan set perubahan. Untuk memperbarui sumber daya yang dilindungi, Anda harus memperbarui kebijakan tumpukan atau menggunakan metode pembaruan langsung.

Execute a change set for nested stacks (console)
Untuk menjalankan set perubahan untuk tumpukan nest (konsol)

  1. Di konsol AWS CloudFormation, di Tumpukan, pilih nama tumpukan yang ingin Anda perbarui. Anda harus memilih nama tumpukan yang berhubungan dengan set perubahan root.

  2. Di panel navigasi, pilih Set Perubahan untuk melihat daftar set perubahan tumpukan.

  3. Pilih nama set perubahan root yang ingin Anda jalankan.

  4. Pada halaman detail set perubahan, pilih Jalankan.

    catatan

    CloudFormation mengeksekusi perubahan yang dijelaskan dalam set perubahan root dan set perubahan bertingkat, jika Diaktifkan untuk set perubahan untuk tumpukan bersarang dipilih selamaMembuat set perubahan proses.

    CloudFormation mulai memperbarui tumpukan dengan segera. CloudFormation Konsol mengarahkan Anda ke tab Acara, di mana Anda dapat memantau kemajuan pembaruan tumpukan.

Untuk menjalankan set perubahan (AWS CLI)

  • Jalankan perintah aws cloudformation execute-change-set.

    Tentukan ID set perubahan pada set perubahan yang ingin Anda jalankan, seperti yang ditunjukkan dalam contoh berikut:

    aws cloudformation execute-change-set --change-set-name arn:aws:cloudformation:us-east-1:123456789012:changeSet/SampleChangeSet/1a2345b6-0000-00a0-a123-00abc0abc000

    Perintah dalam contoh menjalankan set perubahan dengan arn:aws:cloudformation:us-east-1:123456789012:changeSet/SampleChangeSet/1a2345b6-0000-00a0-a123-00abc0abc000 ID.

    Setelah Anda menjalankan perintah tersebut, CloudFormation mulai memperbarui tumpukan, mulai memperbarui tumpukan. Untuk melihat kemajuan tumpukan, gunakan perintah aws cloudformation describe-stacks
