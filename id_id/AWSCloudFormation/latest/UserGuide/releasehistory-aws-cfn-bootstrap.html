Riwayat Rilis untuk CloudFormation Helper Scripts 1.4

Tabel di bawah ini menyediakan akses ke skrip pembantu versi sebelumnya, yang mendukung Python versi 2.7 dan 2.6. Hal ini disediakan untuk kompatibilitas mundur. Kami merekomendasikan penggunaan versi terbaru dari skrip pembantu, yang memerlukan Python 3.4 ke atas.

catatan

Skrip AWS CloudFormation pembantu sudah diinstal sebelumnya pada gambar AMI Amazon Linux. Paket unduhan yang tercantum dalam tabel berlaku untuk distribusi Linux/Unix lainnya dan Microsoft Windows (2008 atau yang lebih baru). Untuk mempelajari cara menggunakan skrip pembantu, lihat CloudFormation referensi skrip pembantu.
