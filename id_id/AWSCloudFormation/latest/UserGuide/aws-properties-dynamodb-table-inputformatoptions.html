AWS::DynamoDB::Table InputFormatOptions

Opsi format untuk data yang diimpor ke tabel target. Ada satu nilai, CsvOption.

Sintaksis

Untuk menyatakan entitas ini di templat AWS CloudFormation Anda, gunakan sintaks berikut:

JSON

{
  "Csv" : Csv
}

YAML

  Csv: 
    Csv

Properti

Csv

Opsi untuk file sumber yang diimpor dalam format CSV. Nilainya adalah Delimiter dan. HeaderList

Wajib: Tidak

Jenis: Csv

Pembaruan membutuhkan: Penggantian
