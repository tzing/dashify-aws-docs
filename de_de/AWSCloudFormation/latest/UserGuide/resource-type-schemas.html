CloudFormation -Ressourcenanbieterschemata

Zusätzlich zur Ressourcenspezifikation CloudFormation stellt die Ressourcenanbieterschemata für die Amazon-Ressourcen nach Region zur Verfügung. Jedes Ressourcenanbieterschema ist eine JSON-formatierte Textdatei, die die Eigenschaften und Attribute einer bestimmten Ressource definiert.

CloudFormation bietet für jede Region eine ZIP-Datei an, die die einzelnen Schemadateien für jede in dieser Region verfügbare Ressource enthält. Informationen zum Herunterladen der Ressourcenanbieterschemas für eine bestimmte Region finden Sie in der folgenden Tabelle. Sie können das Anbieterschema einer Ressource auch mithilfe der CloudFormation -Registrierung anzeigen.

Die Ressourcenverfügbarkeit kann je nach Region variieren. Um die Verfügbarkeit einer Ressource in einer bestimmten Region zu prüfen, sehen Sie unter den Ressourcenanbieterschemas für die jeweilige Region nach.

Weitere Informationen zu Ressourcenanbieterschemata, darunter zu Schemaformat und Definitionen, finden Sie unter Ressourcenanbieterschema im CloudFormation -Befehlszeilenschnittstellen-Benutzerhandbuch.
