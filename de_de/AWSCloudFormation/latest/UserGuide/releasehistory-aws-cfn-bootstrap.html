Versionsverlauf für CloudFormation Hilfsskripte 1.4

Die folgende Tabelle bietet Zugriff auf frühere Versionen der Hilfsskripte, die Python-Version 2.7 und 2.6 unterstützen. Es ist für die Abwärtskompatibilität vorgesehen. Wir empfehlen, die neueste Version der Hilfsskripte zu verwenden, für die Python 3.4 oder höher erforderlich ist.

Anmerkung

Die AWS CloudFormation-Hilfsskripts sind auf Amazon Linux-AMI-Images vorinstalliert. Die in der Tabelle aufgelisteten Download-Pakete gelten für andere Linux/Unix-Verteilungen und Microsoft Windows (2008 oder höher). Informationen zur Verwendung der Hilfsskripts finden Sie unter Referenz für CloudFormation-Hilfsskripts.
