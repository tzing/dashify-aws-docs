AWS::CloudFormation::Interface ParameterGroup

ParameterGroup ist eine Eigenschaft der AWS::CloudFormation::Interface-Ressource, die eine Parametergruppe sowie die in die Gruppe einzuschließenden Parameter definiert.

Syntax

JSON

{
  "Label" : Label,
  "Parameters" : [ String, ... ]
}

YAML

Label: Label
Parameters:
  - String

Eigenschaften

Label

Ein Name für die Parametergruppe.

Required: No

Typ: AWS::CloudFormation::Interface Label

Parameters

Eine Liste von logischen IDs von Parametern, bei denen Groß-/Kleinschreibung unterschieden wird, die in die Gruppe eingeschlossen werden sollen. Parameter müssen bereits im Parameters-Abschnitt der Vorlage definiert sein. Ein Parameter kann nur in eine Parametergruppe eingeschlossen werden.

In der Konsole werden die Parameter, die nicht mit einer Parametergruppe verknüpft werden, in alphabetischer Reihenfolge in der Other parameters-Gruppe aufgeführt.

Required: No

Typ: Liste von Zeichenfolgenwerten
