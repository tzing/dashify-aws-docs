Abrufen einer Vorlage

AWS CloudFormation speichert die Vorlage, die Sie zur Erstellung Ihres Stacks verwenden, als Teil des Stacks. Um die Vorlage von CloudFormation abzurufen, verwenden Sie den Befehl aws cloudformation get-template.

Anmerkung

Der Befehl aws cloudformation get-template gibt die gelöschten Stack-Vorlagen für bis zu 90 Tage nach dem Löschen des Stacks zurück.

Das folgende Beispiel zeigt die Vorlage für den Stack myteststack.

PROMPT> aws cloudformation get-template --stack-name myteststack
{
    "TemplateBody": {
        "AWSTemplateFormatVersion": "2010-09-09",
        "Outputs": {
            "BucketName": {
                "Description": "Name of S3 bucket to hold website content",

                    "Ref": "S3Bucket"
                }
            }
        },
        "Description": "AWS CloudFormation Sample Template S3_Bucket: Sample template showing how to create a publicly accessible S3 bucket. **WARNING** This template creates an S3 bucket.
You will be billed for the AWS resources used if you create a stack from this template.",
        "Resources": {
            "S3Bucket": {
                "Type": "AWS::S3::Bucket",

            }
        }

Die Ausgabe enthält den gesamten Vorlagentext, in Anführungszeichen eingeschlossen.
