Zuweisen und Verknüpfen von Elastic-IP-Adressen mit AWS CloudFormation

Die folgenden Vorlagenausschnitte sind Beispiele für Elastic-IP-Adressen (EIPs) in Amazon EC2. In diesen Beispielen wird die Zuweisung, Verknüpfung und Verwaltung von EIPs für Instances behandelt.

Beispielausschnitte

  • Verknüpfen einer Elastic-IP-Adresse mit einer Amazon-EC2-Instance durch Angabe der IP-Adresse
  • Verknüpfen einer Elastic-IP-Adresse mit einer Amazon-EC2-Instance durch Angabe der Zuweisungs-ID der IP-Adresse

Zuweisen einer Elastic IP-Adresse und deren Verknüpfung mit einer Amazon-EC2-Instance

Mit dem folgenden Ausschnitt wird eine Elastic-IP-Adresse von Amazon EC2 zugewiesen und mithilfe einer AWS::EC2::EIP-Ressource mit einer Amazon-EC2-Instance verknüpft. Eine Elastic-IP-Adresse können Sie aus einem Adresspool von AWS zuweisen oder aus einem Adresspool, der aus einem Bereich öffentlicher IPv4-Adressen erstellt wurde, den Sie für die Verwendung mit Ihren AWS-Ressourcen über Eigene IP-Adressen mitbringen (BYOIP) zu AWS hinzufügen. In diesem Beispiel wird die EIP aus einem Adresspool von AWS zugewiesen.

Weitere Informationen zu Elastic-IP-Adressen finden Sie unter Elastic-IP-Adressen im Benutzerhandbuch von Amazon EC2.

JSON

"ElasticIP": {
    "Type": "AWS::EC2::EIP",
    "Properties": {
        "InstanceId": {
            "Ref": "Ec2Instance"
        }
    }
}

YAML

ElasticIP:
  Type: AWS::EC2::EIP
  Properties:
    InstanceId: !Ref EC2Instance

Verknüpfen einer Elastic-IP-Adresse mit einer Amazon-EC2-Instance durch Angabe der IP-Adresse

Mit dem folgenden Ausschnitt wird eine vorhandene Elastic-IP-Adresse von Amazon EC2 mithilfe einer AWS::EC2::EIPAssociation-Ressource einer EC2-Instance zugewiesen. Dazu müssen Sie zunächst eine Elastic-IP-Adresse für die Verwendung in Ihrem Konto zuweisen. Eine Elastic-IP-Adresse kann mit einer einzelnen Instance verknüpft werden.

JSON

"IPAssoc": {
  "Type": "AWS::EC2::EIPAssociation",
  "Properties": {
    "InstanceId": {
      "Ref": "Ec2Instance"
    },
    "EIP": "192.0.2.0"
  }
}

YAML

IPAssoc:
  Type: AWS::EC2::EIPAssociation
  Properties:
    InstanceId: !Ref EC2Instance
    EIP: 192.0.2.0

Verknüpfen einer Elastic-IP-Adresse mit einer Amazon-EC2-Instance durch Angabe der Zuweisungs-ID der IP-Adresse

Mit dem folgenden Ausschnitt wird eine vorhandene Elastic-IP-Adresse mit einer Amazon-EC2-Instance verknüpft. Dazu wird die Zuweisung-ID mithilfe einer AWS::EC2::EIPAssociation-Ressource angegeben. Eine Zuweisungs-ID wird einer Elastic IP-Adresse nach ihrer Zuweisung zugeordnet.

JSON

"IPAssoc": {
    "Type": "AWS::EC2::EIPAssociation",
    "Properties": {
        "InstanceId": {
            "Ref": "Ec2Instance"
        },
        "AllocationId": "eipalloc-1234567890abcdef0"
    }
}

YAML

IPAssoc:
  Type: AWS::EC2::EIPAssociation
  Properties:
    InstanceId: !Ref EC2Instance
    AllocationId: eipalloc-1234567890abcdef0
