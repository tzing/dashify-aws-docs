AWS CloudFormation -Kontingente

Ihr AWS Konto verfügt über AWS CloudFormation Kontingente, die Sie möglicherweise kennen müssen, wenn Sie Vorlagen erstellen und Stacks erstellen. Durch das Verständnis dieser Kontingente können Sie Einschränkungsfehler vermeiden, bei denen Sie Ihre Vorlagen oder Stacks überarbeiten müssten.

Verfügbarkeit von Funktionen

CloudFormation Möglicherweise sind nicht alle Funktionen von in jeder Region verfügbar. Weitere Informationen zu - AWS Regionen finden Sie in der Tabelle Globale Infrastrukturregionen.

  • In den folgenden Regionen sind Makros zurzeit nicht verfügbar:

    • Asien-Pazifik (Jakarta)

  • Drift-Erkennung ist derzeit in der folgenden Region nicht verfügbar:

    • Asien-Pazifik (Osaka)

  • Die Durchführung von ECS-Blau/Grün-Bereitstellungen CodeDeploy über AWS CloudFormation ist derzeit in den folgenden Regionen nicht verfügbar:

    • Afrika (Kapstadt)

    • Asien-Pazifik (Osaka)

    • Europa (Milan)

StackSets - und -Makros

StackSets unterstützt derzeit nicht das Erstellen oder Aktualisieren von Stack-Sets mit serviceverwalteten Berechtigungen aus Vorlagen, die Makros enthalten. Dazu gehören Transformationen, bei denen es sich um Makros handelt, die von gehostet werden AWS CloudFormation. Weitere Informationen über Makros finden Sie unter Vorlagenmakros.
