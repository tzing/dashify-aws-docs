Erstellen eines Stacks auf der AWS CloudFormation-Konsole

Bevor Sie einen Stack erstellen, müssen Sie über eine Vorlage verfügen, die beschreibt, welche Ressourcen AWS CloudFormation in Ihren Stack einschließt. Weitere Informationen finden Sie unter Arbeiten mit AWS CloudFormation-Vorlagen.

Anmerkung

Für eine Vorschau auf der Konfiguration eines neuen Stacks können Sie einen Änderungssatz verwenden.

Das Erstellen eines Stacks in der CloudFormation-Konsole ist ein einfacher, assistentengesteuerter Prozess, der aus den folgenden Schritte besteht:

  1. Starten des Create Stack-Assistenten.

  2. Auswählen einer Stack-Vorlage

  3. Festlegen der Stack-Parameter

  4. Festlegen von AWS CloudFormation-Stack-Optionen

  5. Prüfen des Stacks

Nach dem Erstellen eines Stacks können Sie den Fortschritt des Stacks verfolgen, seine Ressourcen und Ausgaben anzeigen, den Stack aktualisieren und ihn löschen. Informationen zu diesen Aktionen finden Sie in den zugehörigen Themen.

Starten des Create Stack-Assistenten.

So erstellen Sie einen Stack in der CloudFormation-Konsole

  1. Öffnen Sie die AWS CloudFormation-Konsole unter https://console.aws.amazon.com/cloudformation.

  2. Erstellen Sie einen neuen Stack mit einer der folgenden Optionen:

    • Wählen Sie Create Stack aus. Dies ist die einzige Option, wenn Sie über eine aktuell ausgeführten Stack verfügen.

    • Wählen Sie auf der Seite Stacks Create Stack (Stack erstellen) aus. Diese Option ist nur sichtbar, wenn Sie keine laufenden Stacks haben.

Als Nächstes wählen Sie eine Stack-Vorlage aus.
