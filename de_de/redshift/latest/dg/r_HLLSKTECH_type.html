HLLSKETCH-Typ

Verwenden Sie den Datentyp HLLSKETCH für HyperLogLog Skizzen. Amazon Redshift unterstützt HyperLogLog Skizzendarstellungen, die entweder rudimentär oder ausführlich sind. Skizzen sind zunächst rudimentär und werden ausführlicher, wenn das ausführliche Format effizienter ist, um den verwendeten Speicherbedarf zu minimieren.

Amazon Redshift überträgt eine Sparse- HyperLogLog Skizze automatisch, wenn Skizzen im folgenden JSON-Format importiert, exportiert oder gedruckt werden.

{"logm":15,"sparse":{"indices":[4878,9559,14523],"values":[1,2,1]}}

Amazon Redshift verwendet eine Zeichenfolgendarstellung in einem Base64-Format, um eine ausführliche HyperLogLog Skizze darzustellen.

Amazon Redshift verwendet die folgende Zeichenfolgendarstellung in einem Base64-Format, um eine ausführliche HyperLogLog Skizze darzustellen.

"ABAABA..."

Die maximale Größe eines HLLSKETCH-Objekts beträgt 24 580 Byte, wenn es in RAW-Komprimierung verwendet wird.
