Anzeigen von Amazon-Redshift-Advisor-Empfehlungen

Sie können über die Amazon-Redshift-Konsole, die Amazon-Redshift-API oder auf Amazon-Redshift-Advisor-Empfehlungen zugreifen AWS CLI. Um auf Empfehlungen zuzugreifen, müssen Sie über die -Berechtigung verfügen, die Ihrer IAM-Rolle oder -Identität redshift:ListRecommendations angefügt ist.

Anzeigen von Empfehlungen von Amazon Redshift Advisor in der von Amazon Redshift bereitgestellten Konsole

Sie können Amazon-Redshift-Advisor-Empfehlungen im anzeigen AWS Management Console.

So zeigen Sie Amazon-Redshift-Advisor-Empfehlungen für Amazon-Redshift-Cluster in der Konsole an

  1. Melden Sie sich bei der an AWS Management Console und öffnen Sie die Amazon-Redshift-Konsole unter https://console.aws.amazon.com/redshift/.

  2. Wählen Sie im Navigationsmenü Advisor aus.

  3. Erweitern Sie die jeweilige Empfehlung, um weitere Details anzuzeigen. Auf dieser Seite können Sie Empfehlungen sortieren und gruppieren.

Anzeigen von Amazon-Redshift-Advisor-Empfehlungen mithilfe von Amazon-Redshift-API-Operationen

Sie können Amazon-Redshift-Advisor-Empfehlungen für Amazon-Redshift-Cluster mithilfe der Amazon-Redshift-API auflisten. In der Regel entwickeln Sie und Anwendungen in Ihrer Programmiersprache Ihrer Wahl, um die redshift:ListRecommendations -API mithilfe eines - AWS SDK aufzurufen. Weitere Informationen finden Sie unter ListRecommendations in der Amazon-Redshift-API-Referenz.

Anzeigen von Amazon-Redshift-Advisor-Empfehlungen mithilfe von - AWS Command Line Interface Operationen

Sie können Amazon-Redshift-Advisor-Empfehlungen für Amazon-Redshift-Cluster mithilfe der auflisten AWS Command Line Interface. Weitere Informationen finden Sie unter list-recommendations in der AWS CLI -Befehlsreferenz.
