PG_GET_COLS

Gibt die Spaltenmetadaten für eine Tabellen- oder Ansichtdefinition zurück.

Syntax

pg_get_cols('name')

Argumente

Name

Der Name einer Amazon-Redshift-Tabelle oder -Ansicht. Weitere Informationen finden Sie unter Namen und Kennungen.

Rückgabetyp

VARCHAR

Nutzungshinweise

Die PG_GET_COLS-Funktion gibt für jede Spalte in der Tabelle oder Ansichtdefinition eine Zeile zurück. Die Zeile enthält eine durch Komma getrennte Liste mit dem Schemanamen, dem Beziehungsnamen, dem Spaltennamen, dem Datentyp und der Spaltennummer. Die Formatierung des Ergebnisses von SQL hängt vom verwendeten SQL-Client ab.

Beispiele

Die folgenden Beispiele geben Ergebnisse für eine Ansicht mit dem Namen SALES_VW im Schema public und die Tabellentabelle mit dem Namen sales im Schema zurückmytickit1, die vom Benutzer in der verbundenen Datenbank erstellt werdendev.

Im folgenden Beispiel werden die Spaltenmetadaten für eine Ansicht mit dem Namen zurückgegebenSALES_VW.

select pg_get_cols('sales_vw');

(public,sales_vw,buyerid,integer,4)                        
(public,sales_vw,eventid,integer,5)

(public,sales_vw,qtysold,smallint,7)                       
(public,sales_vw,pricepaid,"numeric(8,2)",8)               
(public,sales_vw,commission,"numeric(8,2)",9)              
(public,sales_vw,saletime,"timestamp without time zone",10)
