Neuer Konfigurationsparameter für die Verwaltung der Cursorgröße

Der Konfigurationsparameter max_cursor_result_set_size definiert den maximalen Umfang der Daten, in Megabyte, die pro Cursor-Ergebnissatz bei einer größeren Abfrage ausgegeben werden können. Dieser Parameterwert beeinflusst auch die Anzahl der gleichzeitigen Cursors für den Cluster, so dass Sie einen Wert konfigurieren können, der die Anzahl der Cursors für Ihren Cluster erhöht oder verringert.

Weitere Informationen finden Sie unter DECLARE in diesem Leitfaden und unter Konfigurieren der maximalen Größe eines Cursor-Ergebnissatzes im Amazon-Redshift-Verwaltungshandbuch.
