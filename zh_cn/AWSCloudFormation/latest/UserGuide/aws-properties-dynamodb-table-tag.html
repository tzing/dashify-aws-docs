AWS::DynamoDB::Table Tag

描述标签。标签是键值对。您最多可以向单个 DynamoDB 表添加 50 个标签。

AWS-assigned 的标签名称和值会自动分配用户无法分配aws:的前缀。 AWS分配的标签名称不计入 50 的标签限制。用户分配的标签名称在成本分配user:报告中带有前缀。您不能回溯标签的应用日期。

有关为 DynamoDB 资源添加标签的概述，请参阅亚马逊 DynamoDB 开发者指南中的为 DynamoDB 添加标签。

语法

要在 AWS CloudFormation 模板中声明此实体，请使用以下语法：

JSON

{
  "Key" : String,
  "Value" : String
}

YAML

  Key: String
  Value: String

属性

Key

标签的密钥。标签键区分大小写。每个 DynamoDB 表最多只能有一个具有相同密钥的标签。如果您尝试添加现有标签（相同的密钥），则现有标签值将更新为新值。

必需：是

类型：字符串

最小值：1

最大值：128

更新要求：没有中断

Value

标签的值。标签值区分大小写，可以为空。

必需：是

类型：字符串

最小值：0

最大值：256

更新要求：没有中断
