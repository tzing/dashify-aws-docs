AWS::DynamoDB::Table ImportSourceSpecification

指定从 S3 桶源导入到表的数据的属性。

语法

要在 AWS CloudFormation 模板中声明此实体，请使用以下语法：

JSON

{
  "InputCompressionType" : String,
  "InputFormat" : String,
  "InputFormatOptions" : InputFormatOptions,
  "S3BucketSource" : S3BucketSource
}

YAML

  InputCompressionType: String
  InputFormat: String
  InputFormatOptions: 
    InputFormatOptions
  S3BucketSource: 
    S3BucketSource

属性

InputCompressionType

要对来自导入表的输入使用的压缩类型。

必需：否

类型：字符串

允许的值：GZIP | ZSTD | NONE

更新要求：替换

InputFormat

源数据的格式。ImportFormat 的有效值为 CSV、DYNAMODB_JSON 或 ION。

必需：是

类型：字符串

允许的值：DYNAMODB_JSON | ION | CSV

更新要求：替换

InputFormatOptions

指定如何设置输入格式的其他属性。

必需：否

类型:InputFormatOptions

更新要求：替换

S3BucketSource

为导入提供源的 S3 桶。

必需：是

类型：S3 BucketSource

更新要求：替换
