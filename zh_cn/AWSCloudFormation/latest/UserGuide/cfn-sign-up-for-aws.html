注册 AWS 账户和定价

您必须首先注册 AWS 账户，然后才能使用 AWS CloudFormation 或任何亚马逊云科技服务。

要注册AWS 账户

  1. 打开 https://portal.aws.amazon.com/billing/signup。

  2. 按照屏幕上的说明进行操作。

    在注册时，将接到一通电话，要求使用电话键盘输入一个验证码。

    注册AWS 账户时，系统将会创建AWS 账户根用户。根用户有权访问该账户中的所有AWS 服务和资源。作为一种安全最佳实践，请为管理用户分配管理访问权限，并且只使用根用户执行需要根用户访问权限的任务。

在注册 AWS 账户 之后，您可以通过 AWS Management Console、AWS CloudFormation API 或 AWS CLI 使用 AWS CloudFormation。

定价

AWS CloudFormation 是一项免费服务；不过，您的堆栈中包含的每种 AWS 资源按其当前费率进行收费。有关 AWS 定价的更多信息，请转至 http://aws.amazon.com 上每种产品的详细信息页面。
