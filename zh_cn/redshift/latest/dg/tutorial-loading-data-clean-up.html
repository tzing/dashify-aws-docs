步骤 7：清理资源

只要您的集群正在运行，就将继续产生费用。完成本教程后，您应按照《Amazon Redshift 入门指南》中的步骤 5：撤消访问权限并删除示例集群中的步骤操作，以使您的环境返回上一个状态。

如果您希望保留集群，但恢复 SSB 表使用的存储空间，请运行以下命令。

drop table part;
drop table supplier;
drop table customer;
drop table dwdate;
drop table lineorder;
