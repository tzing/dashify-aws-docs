可进行数据共享的区域（预览版）

以下是通过 PREVIEW_2023 版本中公开预览版 Amazon Redshift 数据共享功能写入多数据仓库的预发行文档。文档和特征都可能会更改。我们建议您只在测试集群中使用此功能，而不要在生产环境中使用。有关预览条款和条件，请参阅 AWS 服务条款中的测试版服务参与。

如果您尚未在预览版跟踪上创建数据共享，请前往 AWS 账户内或跨账户共享读取和写入数据开始使用。

以下区域可进行数据共享（预览版）：

  • 美国东部（弗吉尼亚州北部）(us-east-1)

  • 美国东部（俄亥俄州）(us-east-2)

  • 美国西部（俄勒冈州）(us-west-2)

  • 亚太地区（东京）(ap-northeast-1)

  • 欧洲地区（爱尔兰）(eu-west-1)

  • 欧洲地区（斯德哥尔摩）(eu-north-1)
