表列

SVV_IDENTITY_PROVIDERS

SVV_IDENTITY_PROFERS 视图返回身份提供商的名称和其他属性。有关如何创建身份提供商角色的更多信息，请参阅 CREATE IDENTITY PROVIDER。

SVV_IDENTITY_PROVIDERS 只对超级用户可见。有关更多信息，请参阅系统表和视图中的数据可见性。

表列

列名称 数据类型 描述
uid 整数 已注册的身份提供商的唯一 ID。
名称 text 身份提供商名称。
类型 text 身份提供商类型。
instanceid text 相同类型的实例之间的唯一区分元素。
namespc text 身份提供商的命名空间前缀。
params text 带有身份提供商参数的 JSON 对象。
已启用 布尔 指示身份提供商是否已启用。

示例查询

要查看身份提供商属性，请在创建身份提供商之后，运行类似以下内容的查询。

SELECT name, type, instanceid, namespc, params, enabled 
FROM svv_identity_providers 
ORDER BY 1;
