适用于 RBAC 的 ALTER DEFAULT PRIVILEGES

使用 ALTER DEFAULT PRIVILEGES 来定义默认访问权限集，这些权限将应用于指定的用户未来创建的对象。默认情况下，用户只能更改他们自己的默认访问权限。使用 RBAC，您可以设置角色的默认访问权限。有关更多信息，请参阅 ALTER DEFAULT PRIVILEGES 命令。

与系统权限类似，您可以通过 RBAC 将数据库对象权限分配给角色。然后，您可以为用户分配角色、向用户授予系统权限和/或数据库权限。
