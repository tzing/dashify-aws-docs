기계 번역으로 제공되는 번역입니다. 제공된 번역과 원본 영어의 내용이 상충하는 경우에는 영어 버전이 우선합니다.

AWS::DynamoDB::Table S3BucketSource

가져오는 S3 버킷입니다.

조건

AWS CloudFormation 템플릿에서 이 개체를 선언하려면 다음 구문을 사용합니다.

JSON

{
  "S3Bucket" : String,
  "S3BucketOwner" : String,
  "S3KeyPrefix" : String
}

YAML

  S3Bucket: String
  S3BucketOwner: String
  S3KeyPrefix: String

속성

S3Bucket

가져오는 S3 버킷입니다.

필수 항목 여부: 예

유형: 문자열

Pattern: ^[a-z0-9A-Z]+[\.\-\w]*[a-z0-9A-Z]+$

Maximum: 255

업데이트 필요: 대체

S3BucketOwner

가져오는 S3 버킷의 계정 번호입니다. 요청자가 버킷을 소유한 경우 이는 선택 사항입니다.

필수 항목 여부: 아니요

유형: 문자열

Pattern: [0-9]{12}

업데이트 필요: 대체

S3KeyPrefix

가져오는 모든 S3 객체가 공유하는 키 접두사입니다.

필수 항목 여부: 아니요

유형: 문자열

Maximum: 1024

업데이트 필요: 대체
