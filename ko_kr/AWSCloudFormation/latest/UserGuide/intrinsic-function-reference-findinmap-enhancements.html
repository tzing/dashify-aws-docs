Fn::FindInMap enhancements

AWS CloudFormation 템플릿에 AWS::LanguageExtensions 변환을 추가할 때 내장 함수를 사용하여 Fn::FindInMap의 필드를 정의할 수 있습니다. 매핑을 찾을 수 없는 경우 새 선택적 필드를 사용하여 기본값을 반환할 수도 있습니다.

AWS::LanguageExtensions 변환에 대한 자세한 내용은 AWS::LanguageExtensions 변환 섹션을 참조하세요.

Fn::FindInMap 내장 함수에 대한 자세한 내용은 Fn::FindInMap 섹션을 참조하세요.

선언

JSON

{ "Fn::FindInMap" : [
    "MapName",
    "TopLevelKey",
    "SecondLevelKey",
    {"DefaultValue": "DefaultValue"}
  ]
}

YAML

전체 함수 이름의 구문:

Fn::FindInMap:
  - MapName
  - TopLevelKey
  - SecondLevelKey
  - DefaultValue: DefaultValue

짧은 형식의 구문:

!FindInMap
  - MapName
  - TopLevelKey
  - SecondLevelKey
  - DefaultValue: DefaultValue

파라미터

DefaultValue

TopLevelKey 및/또는 SecondLevelKey를 MapName 맵에서 찾을 수 없는 경우 Fn::FindInMap이 확인할 값입니다. 이 필드는 선택 사항입니다.

모든 파라미터 MapName, TopLevelKey, SecondLevelKey 및 DefaultValue는 변환 중 유효한 값으로 확인할 수 있는 한 내장 함수일 수 있습니다.

예

다음은 내장 함수를 사용하여 최상위 키를 정의하는 예제입니다.

JSON

{
  //...
  "Transform": "AWS::LanguageExtensions",
  //...
      "Fn::FindInMap": [
        "MyMap",
        {
          "Fn::Select": [
            0,
            {
              "Fn::Split": [
                "|",
                { "Ref": "InputKeys" }
              ]
            }
          ]
        },
        "SecondKey"
      ]
//...
}

YAML

Transform: 'AWS::LanguageExtensions'
#...
    !FindInMap: [MyMap, !Select [0, !Split [|, !Ref InputKeys]], SecondKey]
#...

다음은 기본값 사용의 예입니다.

JSON

{
  //...
    "Transform": "AWS::LanguageExtensions",
    //...
    "Fn::FindInMap": [
      "InstanceConfiguration",
      { "Ref": "AWS::Region" },
      "Type",
      { "DefaultValue": "m5.small" }
    ]
  //...
}

YAML

Transform: 'AWS::LanguageExtensions'
#...
    !FindInMap 
        - 'InstanceConfiguration'
        - !Ref 'AWS::Region'
        - 'Type'
        - DefaultValue: m5.small
#...

지원되는 함수

Fn::FindInMap: 향상 기능의 파라미터에 다음 함수를 사용할 수 있습니다.

  • Fn::FindInMap

  • Fn::Join

  • Fn::Sub

  • Fn::If

  • Fn::Select

  • Fn::Length

  • Fn::ToJsonString

  • Fn::Split - 기본값으로 사용되지 않는 한 Fn::Split은 Fn::Join 또는 Fn::Select와 같이 문자열을 생성하는 내장 함수와 함께 사용해야 합니다.

  • Ref
