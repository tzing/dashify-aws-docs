AWS::DynamoDB::GlobalTable PointInTimeRecoverySpecification

Representa as configurações usadas para habilitar a recuperação point-in-time.

Sintaxe

Para declarar esta entidade no modelo do AWS CloudFormation, use a seguinte sintaxe:

JSON

{
  "PointInTimeRecoveryEnabled" : Boolean
}

YAML

  PointInTimeRecoveryEnabled: Boolean

Propriedades

PointInTimeRecoveryEnabled

Indica se a recuperação point-in-time está habilitada (verdadeiro) ou desabilitada (falso) na tabela.

Obrigatório: Não

Tipo: booliano

Requisitos da atualização: sem interrupção
