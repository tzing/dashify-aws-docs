Esquemas do provedor de recursos do CloudFormation

Além da especificação do recurso, o CloudFormation disponibiliza os esquemas do provedor de recursos para os recursos da Amazon por região. Cada esquema do provedor de recursos é um arquivo de texto formatado em JSON que define as propriedades e os atributos de um recurso específico.

Para cada região, o CloudFormation oferece um arquivo zip que contém os arquivos de esquema individuais para cada recurso disponível na região. Para fazer download dos esquemas do provedor de recursos em uma região específica, consulte a tabela a seguir. Você também pode visualizar o esquema do provedor de um recurso usando o registro do CloudFormation.

A disponibilidade de recursos pode variar de acordo com a região. Para verificar a disponibilidade de um recurso em uma determinada região, consulte os esquemas do provedor de recursos da região.

Para saber mais sobre esquemas de provedores de recursos, incluindo o formato e as definições do esquema, consulte Esquema do provedor de recursos no Guia do usuário da interface de linhas de comando do CloudFormation.
