AWS::DynamoDB::Table ImportSourceSpecification

Especifica as propriedades dos dados que estão sendo importados da origem do bucket do S3 para a tabela.

Sintaxe

Para declarar esta entidade no modelo do AWS CloudFormation, use a seguinte sintaxe:

JSON

{
  "InputCompressionType" : String,
  "InputFormat" : String,
  "InputFormatOptions" : InputFormatOptions,
  "S3BucketSource" : S3BucketSource
}

YAML

  InputCompressionType: String
  InputFormat: String
  InputFormatOptions: 
    InputFormatOptions
  S3BucketSource: 
    S3BucketSource

Propriedades

InputCompressionType

Tipo de compactação a ser usado na entrada proveniente da tabela importada.

Obrigatório: não

Tipo: string

Valores permitidos: GZIP | ZSTD | NONE

Requisitos da atualização: substituição

InputFormat

O formato dos dados de origem. Os valores válidos para ImportFormat são CSV, DYNAMODB_JSON ou ION.

Obrigatório: Sim

Tipo: string

Valores permitidos: DYNAMODB_JSON | ION | CSV

Requisitos da atualização: substituição

InputFormatOptions

Propriedades adicionais que especificam como a entrada é formatada,

Obrigatório: não

Tipo: InputFormatOptions

Requisitos da atualização: substituição

S3BucketSource

O bucket do S3 que fornece a origem para a importação.

Obrigatório: Sim

Tipo: S3 BucketSource

Requisitos da atualização: substituição
