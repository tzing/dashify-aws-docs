AWS::DynamoDB::Table InputFormatOptions

As opções de formato para os dados que foram importados para a tabela de destino. Há um valor, CsvOption.

Sintaxe

Para declarar esta entidade no modelo do AWS CloudFormation, use a seguinte sintaxe:

JSON

{
  "Csv" : Csv
}

YAML

  Csv: 
    Csv

Propriedades

Csv

As opções para arquivos de origem importados no formato CSV. Os valores são Delimitador e. HeaderList

Obrigatório: não

Tipo: Csv

Requisitos da atualização: substituição
