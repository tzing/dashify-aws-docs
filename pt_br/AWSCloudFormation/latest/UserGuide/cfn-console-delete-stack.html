Excluir uma pilha no console do AWS CloudFormation

Para excluir uma pilha do

  1. Abra o console do AWS CloudFormation em https://console.aws.amazon.com/cloudformation.

  2. Na página Pilhas no console do CloudFormation, selecione a pilha que deseja excluir. A pilha deve estar em execução no momento.

  3. No painel de detalhes da pilha, escolha Delete (Excluir).

  4. Selecione Delete stack (Excluir pilha) quando solicitado.

    nota

    O operação de exclusão da pilha não pode ser interrompida uma vez que a ação já tenha começado. A pilha continua para o estado DELETE_IN_PROGRESS.

    Quando a exclusão da pilha for concluída, a pilha estará no estado DELETE_COMPLETE. Por padrão, as pilhas no estado DELETE_COMPLETE não são exibidas no console do CloudFormation. Para exibir as pilhas excluídas, é necessário alterar o filtro de visualização de pilhas, conforme descrito em Visualizar pilhas excluídas no console do AWS CloudFormation.

    Se houver falha ao excluir, a pilha estará no estado DELETE_FAILED. Para obter soluções, consulte o tópico de solução de problemas Falha na exclusão da pilha.

Para obter informações sobre proteção contra exclusão acidental de pilhas, consulte Proteger uma pilha contra exclusão.
