Histórico de lançamentos de scripts Helper CloudFormation 1.4

A tabela abaixo fornece acesso a versões anteriores dos scripts auxiliares, que oferecem suporte ao Python versões 2.7 e 2.6. É fornecido para compatibilidade com versões anteriores. Recomendamos usar a versão mais recente dos scripts auxiliares, que exigem o Python 3.4 ou superior.

nota

Os scripts auxiliares do AWS CloudFormation são pré-instalados nas imagens de AMI do Amazon Linux. Os pacotes de download listados na tabela se aplicam a outras distribuições do Linux/Unix e Microsoft Windows (2008 ou posterior). Para aprender a usar os scripts auxiliares, consulte CloudFormation referência de scripts auxiliares.
