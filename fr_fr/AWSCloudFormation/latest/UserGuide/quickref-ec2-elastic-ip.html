Attribuer et associer des adresses IP Elastic avec AWS CloudFormation

Les extraits de modèle suivants sont des exemples relatifs aux adresses IP Elastic (EIP) dans Amazon EC2. Ces exemples couvrent l'attribution, l'association et la gestion des adresses IP Elastic pour vos instances.

Exemples d'extraits
  • Attribuer une adresse IP Elastic et l'associer à une instance Amazon EC2
  • Associer une adresse IP Elastic à une instance Amazon EC2 en spécifiant l'adresse IP
  • Associer une adresse IP Elastic à une instance Amazon EC2 en spécifiant l'ID d'allocation de l'adresse IP

Attribuer une adresse IP Elastic et l'associer à une instance Amazon EC2

L'extrait suivant attribue une adresse IP Elastic (EIP) Amazon EC2 et l'associe à une instance Amazon EC2 à l'aide d'une ressource AWS::EC2::EIP. Vous pouvez attribuer une adresse IP Elastic à partir d'un groupe d'adresses détenu par AWS ou d'un groupe d'adresses créé à partir d'une plage d'adresses IPv4 publiques que vous apportez dans AWS pour l'utiliser avec vos ressources AWS à l'aide de la fonction Apportez votre propre adresse IP (BYOIP). Dans cet exemple, l'EIP est attribuée à partir du groupe d'adresses appartenant à AWS.

Pour plus d'informations sur les adresses IP Elastic, consultez Adresses IP Elastic dans le Guide de l'utilisateur Amazon EC2.

JSON

"ElasticIP": {
    "Type": "AWS::EC2::EIP",
    "Properties": {
        "InstanceId": {
            "Ref": "Ec2Instance"
        }
    }
}

YAML

ElasticIP:
  Type: AWS::EC2::EIP
  Properties:
    InstanceId: !Ref EC2Instance

Associer une adresse IP Elastic à une instance Amazon EC2 en spécifiant l'adresse IP

L'extrait ci-après associe une adresse IP Elastic Amazon EC2 existante à une instance EC2 à l'aide d'une ressource AWS::EC2::EIPAssociation. Vous devez d'abord attribuer une adresse IP Elastic à utiliser dans votre compte. Une adresse IP Elastic peut être associée à une seule instance.

JSON

"IPAssoc": {
  "Type": "AWS::EC2::EIPAssociation",
  "Properties": {
    "InstanceId": {
      "Ref": "Ec2Instance"
    },
    "EIP": "192.0.2.0"
  }
}

YAML

IPAssoc:
  Type: AWS::EC2::EIPAssociation
  Properties:
    InstanceId: !Ref EC2Instance
    EIP: 192.0.2.0

Associer une adresse IP Elastic à une instance Amazon EC2 en spécifiant l'ID d'allocation de l'adresse IP

L'extrait suivant associe une adresse IP Elastic existante à une instance Amazon EC2 en spécifiant l'ID d'allocation à l'aide d'une ressource AWS::EC2::EIPAssociation. Un ID d'allocation est attribué à une adresse IP Elastic lors de l'allocation d'adresse IP élastique.

JSON

"IPAssoc": {
    "Type": "AWS::EC2::EIPAssociation",
    "Properties": {
        "InstanceId": {
            "Ref": "Ec2Instance"
        },
        "AllocationId": "eipalloc-1234567890abcdef0"
    }
}

YAML

IPAssoc:
  Type: AWS::EC2::EIPAssociation
  Properties:
    InstanceId: !Ref EC2Instance
    AllocationId: eipalloc-1234567890abcdef0
