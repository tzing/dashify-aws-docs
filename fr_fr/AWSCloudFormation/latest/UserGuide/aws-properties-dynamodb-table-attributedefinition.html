AWS::DynamoDB::Table AttributeDefinition

Représente un attribut permettant de décrire le schéma de la table et des index.

Syntaxe

Pour déclarer cette entité dans votre AWS CloudFormation modèle, utilisez la syntaxe suivante :

JSON

{
  "AttributeName" : String,
  "AttributeType" : String
}

YAML

  AttributeName: String
  AttributeType: String

Propriétés

AttributeName

Nom de l'attribut.

Obligatoire : oui

Type : chaîne

Minimum : 1

Maximum : 255

Requis pour la mise à jour : aucune interruption

AttributeType

Type de données de l'attribut, où :

  • S – L'attribut est de type chaîne

  • N – L'attribut est de type nombre

  • B – L'attribut est de type binaire

Obligatoire : oui

Type : chaîne

Valeurs autorisées : S | N | B

Requis pour la mise à jour : aucune interruption
