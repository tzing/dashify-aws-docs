Historique des versions de CloudFormation Helper Scripts 1.4

Le tableau ci-dessous donne accès aux versions précédentes des scripts d'assistant, qui prennent en charge Python versions 2.7 et 2.6. Il est fourni à des fins de rétrocompatibilité. Nous vous recommandons d'utiliser la dernière version des scripts d'assistant, qui nécessitent Python 3.4 ou une version supérieure.

Note

Les scripts d'assistant AWS CloudFormation sont préinstallés sur les images AMI Amazon Linux. Les packages de téléchargement répertoriés dans le tableau s'appliquent à d'autres distributions Linux/Unix et Microsoft Windows (2008 ou version ultérieure). Pour savoir comment utiliser les scripts d'assistant, consultez Référence des scripts d'assistant CloudFormation.
