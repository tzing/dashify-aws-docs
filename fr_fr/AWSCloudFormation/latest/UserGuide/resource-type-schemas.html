CloudFormation schémas de fournisseurs de ressources

Outre la spécification des ressources, CloudFormation met à disposition les schémas des fournisseurs de ressources pour les ressources Amazon par région. Chaque schéma de fournisseur de ressources est un fichier texte au format JSON qui définit les propriétés et les attributs d'une ressource spécifique.

Pour chaque région, CloudFormation propose un fichier zip contenant les fichiers de schéma individuels pour chaque ressource disponible dans cette région. Pour télécharger les schémas de fournisseur de ressources pour une région spécifique, reportez-vous au tableau suivant. Vous pouvez également afficher le schéma de fournisseur d'une ressource à l'aide du registre CloudFormation .

La disponibilité de la ressource peut varier en fonction de la région. Pour vérifier la disponibilité d'une ressource dans une région donnée, veuillez consulter les schémas du fournisseur de ressources pour cette région.

Pour de plus amples informations sur les schémas de fournisseur de ressources, y compris le format et les définitions de schéma, veuillez consulter Schéma de fournisseur de ressources dans le Guide de l'utilisateur de l'interface de ligne de commande CloudFormation .
