Modèle AWS Lambda

Le modèle suivant utilise une fonction AWS Lambda (Lambda) et une ressource personnalisée pour ajouter un nouveau groupe de sécurité à une liste de groupes de sécurité existants. Cette fonction s'avère utile lorsque vous voulez créer dynamiquement une liste de groupes de sécurité, afin d'y inclure des groupes de sécurité à la fois nouveaux et existants. Par exemple, vous pouvez transmettre une liste des groupes de sécurité existants sous forme de valeur de paramètre, ajouter la nouvelle valeur à la liste, puis associer toutes vos valeurs à une instance EC2. Pour plus d'informations sur le type de ressource de la fonction Lambda, consultez. AWS::Lambda::Function

Dans cet exemple, lorsque AWS CloudFormation crée la ressource personnalisée AllSecurityGroups, AWS CloudFormation appelle la fonction Lambda AppendItemToListFunction. AWS CloudFormation transmet la liste des groupes de sécurité existants et un nouveau groupe de sécurité (NewSecurityGroup) à la fonction, qui ajoute le nouveau groupe de sécurité à la liste, puis renvoie la liste modifiée. AWS CloudFormation utilise la liste modifiée pour associer tous les groupes de sécurité à la ressource MyEC2Instance.
