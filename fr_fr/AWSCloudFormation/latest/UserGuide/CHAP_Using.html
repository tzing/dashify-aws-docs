Procédures

Les modèles sont des fichiers au format JSON ou YAML qui décrivent les ressources AWS que vous souhaitez inclure ou mettre à jour dans vos piles AWS CloudFormation. Pour créer des modèles, vous pouvez utiliser AWS CloudFormation Designer ou un éditeur de texte.

Les procédures suivantes vous aident à créer des exemples de modèles AWS CloudFormation via AWS CloudFormation Designer et en texte brut.

Rubriques
  • Procédure de création d'un serveur web de base avec AWS CloudFormation Designer
  • Procédure de modification du modèle d'une pile via AWS CloudFormation Designer
  • Procédure : appairage avec un VPC dans un autre Compte AWS
  • Procédure pas à pas : création de références à des sorties de ressources dans une autre pile AWS CloudFormation
  • Procédure : créer une application redimensionnée et équilibrée de charge
  • Déploiement d'applications sur Amazon EC2 avec AWS CloudFormation
  • Création de conditions Wait dans un gabarit
