StackSet Détails de l'événement de changement de statut

Vous trouverez ci-dessous les champs détaillés relatifs aux événements StackSet de changement de statut.

Les detail-type champs source et sont inclus car ils contiennent des valeurs spécifiques pour les AWS CloudFormation événements.

{
  . . .,
  "detail-type": "CloudFormation StackSet Status Change",
  "source": "aws.cloudformation",
  . . .,
  "detail": {
    "stack-set-arn" : "string",
    "status-details": {
        "status":"string"
    }
  }
}
detail-type

Identifie le type d'événement.

Pour les événements d' StackSet état, cette valeur estCloudFormation StackSet Status Change.

source

Identifie le service qui a généré l'événement. Pour les CloudFormation événements, cette valeur estaws.cloudformation.

detail

Un objet JSON qui contient des informations sur l'événement. Le service qui génère l'événement détermine le contenu de ce champ.

Pour les événements d' StackSet état, ces données incluent :

stack-set-arn

Amazon Resource Name (ARN) associé à l'ensemble de piles.

status-details
status

Le StackSet statut.

Valeurs valides : ACTIVE | DELETED

Exemple : événement StackSet de changement de statut

Voici un exemple d'événement de changement de StackSet statut. Cet événement indique qui CloudFormation a supprimé l'ensemble de piles spécifié.

  "account": "111122223333",
  "time": "2021-09-23T17:06:18Z",
  "region": "us-east-1",

    "arn:aws:cloudformation:us-east-1:111122223333:stackset/test12345:3f3a3fbe-c937-4eb3-a87d-e36a0af3f663"

  "detail": {
    "stack-set-arn" : "arn:aws:cloudformation:us-east-1:111122223333:stackset/test12345:3f3a3fbe-c937-4eb3-a87d-e36a0af3f663",
    "status-details": {
        "status":"DELETED"
    }
  }
}
