SYS_EXTERNAL_QUERY_ERROR

Vous pouvez interroger la vue système SYS_EXTERNAL_QUERY_ERROR pour obtenir des informations sur les erreurs d’analyse Redshift Spectrum. SYS_EXTERNAL_QUERY_ERROR affiche un exemple d’erreurs journalisées. La valeur par défaut est de 10 entrées par requête.

SYS_EXTERNAL_QUERY_ERROR est visible pour tous les utilisateurs. Les super-utilisateurs peuvent voir toutes les lignes, tandis que les utilisateurs standard peuvent voir uniquement leurs propres données. Pour plus d'informations, consultez Visibilité des données dans les tables et vues système.

Colonnes de la table

Nom de la colonne Type de données Description
user_id entier Identifiant de l’utilisateur qui a généré cette ligne.
query_id bigint Identifiant de la requête qui a généré cette ligne.
file_location char(256) L’emplacement des données à interroger.
rowid char(2100) Emplacement de l’erreur dans le fichier. Les parties rowid sont séparées par : (deux-points), et des parties supplémentaires peuvent être ajoutées plus tard. 
row_offset:row_group:row_id
Un row_offset correspond au décalage (en octets) de la ligne dans le fichier et est défini sur -1 pour les formats de fichier non pris en charge. Une table est divisée en row_groups, et chaque groupe possède des lignes avec des row_ids distincts.
column_name char(127) Le nom de la colonne renvoyée par la requête.
original_value char(1024) Valeur d’origine interrogée.
modified_value char(1024) Valeur modifiée renvoyée en fonction de l’option de configuration de gestion des données spécifiée dans la requête.
déclencheur char(128) Option de gestion des données spécifiée dans la requête.
action char(128) Action associée à l’option de gestion des données spécifiée dans la requête.
action_value char(128) Paramètre de valeur d’action associé à l’option de traitement des données spécifiée dans la requête.
error_code entier Code de résultat de l’option de traitement des données spécifiée dans la requête.

Exemple de requête

La requête suivante renvoie la liste des lignes pour lesquelles des opérations de traitement des données ont été effectuées. 

SELECT * FROM sys_external_query_error;
