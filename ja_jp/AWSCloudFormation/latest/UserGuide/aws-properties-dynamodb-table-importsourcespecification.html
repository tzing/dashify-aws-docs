AWS::DynamoDB::Table ImportSourceSpecification

S3 バケットソースからテーブルにインポートされるデータのプロパティを指定します。

[Syntax (構文)]

AWS CloudFormation テンプレートでこのエンティティを宣言するには、以下の構文を使用します。

JSON

{
  "InputCompressionType" : String,
  "InputFormat" : String,
  "InputFormatOptions" : InputFormatOptions,
  "S3BucketSource" : S3BucketSource
}

YAML

  InputCompressionType: String
  InputFormat: String
  InputFormatOptions: 
    InputFormatOptions
  S3BucketSource: 
    S3BucketSource

プロパティ

InputCompressionType

インポートされたテーブルからの入力に使用される圧縮タイプ。

必須: いいえ

型: 文字列

許可される値: GZIP | ZSTD | NONE

更新に伴う要件: 置換

InputFormat

ソースデータの形式。ImportFormat の有効値は、CSV、DYNAMODB_JSON、または ION です。

必須: はい

型: 文字列

許可される値: DYNAMODB_JSON | ION | CSV

更新に伴う要件: 置換

InputFormatOptions

入力のフォーマット方法を指定する追加のプロパティ、

必須: いいえ

タイプ:InputFormatOptions

更新に伴う要件: 置換

S3BucketSource

インポートのソースを提供する S3 バケット。

必須: はい

タイプ:S3 BucketSource

更新に伴う要件: 置換
