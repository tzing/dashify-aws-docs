AWS::DynamoDB::Table S3BucketSource

インポート元の S3 バケット。

構文

AWS CloudFormation テンプレートでこのエンティティを宣言するには、以下の構文を使用します。

JSON

{
  "S3Bucket" : String,
  "S3BucketOwner" : String,
  "S3KeyPrefix" : String
}

YAML

  S3Bucket: String
  S3BucketOwner: String
  S3KeyPrefix: String

プロパティ

S3Bucket

インポート元の S3 バケット。

必須: はい

タイプ: 文字列

パターン: ^[a-z0-9A-Z]+[\.\-\w]*[a-z0-9A-Z]+$

最大: 255

更新に伴う要件: 置換

S3BucketOwner

インポート元の S3 バケットのアカウント番号。バケットがリクエスタによって所有されている場合、これはオプションです。

必須: いいえ

タイプ: 文字列

パターン: [0-9]{12}

更新に伴う要件: 置換

S3KeyPrefix

インポートされるすべての S3 オブジェクトで共有されるキープレフィックス。

必須: いいえ

タイプ: 文字列

最大: 1024

更新に伴う要件: 置換
