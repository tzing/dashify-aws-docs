取用者帳戶管理員動作

如果您是取用者帳戶管理員，請依照下列步驟執行：

若要將從其他帳戶共用的一或多個資料共用與整個 AWS 帳戶 或您帳戶中的特定叢集命名空間建立關聯，請使用 Amazon Redshift 主控台。

登入 https://console.aws.amazon.com/redshift/。然後，將從其他帳戶共用的一或多個資料共用與整個 AWS 帳戶 或您帳戶中的特定叢集命名空間建立關聯。如需詳細資訊，請參閱建立資料共用的關聯。

與 AWS 帳戶 或特定叢集命名空間建立關聯之後，資料共用就可供使用。您也可以隨時變更資料共用關聯。將個別叢集命名空間的關聯變更為 AWS 帳戶 時，Amazon Redshift 會以 AWS 帳戶 資訊覆寫叢集命名空間。將 AWS 帳戶 中的關聯變更為特定叢集命名空間時，Amazon Redshift 會以叢集命名空間資訊覆寫 AWS 帳戶 資訊。帳戶中的所有叢集命名空間都可以存取資料。
