系統目錄存放結構描述中繼資料，例如資料表和資料欄的相關資訊。系統目錄資料表有 PG 字首。

Amazon Redshift 使用者可存取標準 PostgreSQL 目錄資料表。如需 PostgreSQL 系統目錄的詳細資訊，請參閱 PostgreSQL system tables
