SVV 中繼資料檢視

SVV 檢視是 Amazon Redshift 中的系統檢視，其中包含資料庫物件的相關資訊。

注意

如果資料庫回應因任何原因而失敗，Amazon Redshift 會報告 WARNING，而非ERROR。當您在資料共用中查詢物件時，Amazon Redshift 不會傳送 ERROR 訊息。
