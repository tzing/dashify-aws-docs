Amazon Redshift 和 PostgreSQL JDBC 和 ODBC

由於 Amazon Redshift 是以 PostgreSQL 為基礎，因此我們先前建議使用 JDBC4 Postgresql 驅動程式版本 8.4.703 和 psqlODBC 版本 9.x 驅動程式。如果您目前正在使用這些驅動程式，我們建議今後移至新的 Amazon Redshift 專屬驅動程式。如需有關驅動程式和設定連線的詳細資訊，請參閱《Amazon Redshift 管理指南》中的適用於 Amazon Redshift 的 JDBC 和 ODBC 驅動程式。

若要避免使用 JDBC 擷取大型資料集時發生用戶端 out-of-memory 錯誤，您可以設定 JDBC 擷取大小參數，讓用戶端能夠批次擷取資料。如需詳細資訊，請參閱設定 JDBC 擷取大小參數。

Amazon Redshift 無法辨識 JDBC maxRows 參數。請改為指定 LIMIT 子句來限制結果集。您也可以使用 OFFSET 子句，來跳到結果集中的指定起點。
