CURRENT_SCHEMA

傳回搜尋路徑前面的結構描述名稱。此結構描述將用於建立時未指定目標結構描述的任何資料表或其他具名物件。

語法

注意

這是領導者節點函數。此函數在參考使用者建立的資料表、STL 或 STV 系統資料表，或 SVV 或 SVL 系統檢視時會傳回錯誤。

current_schema()

傳回類型

CURRENT_SCHEMA 傳回 CHAR 或 VARCHAR 字串。

範例

下列查詢傳回目前結構描述：

select current_schema();
