AWS CloudFormation StackSets 範例範本

本節所涵蓋的部分範例 AWS CloudFormation 範本連結，有助於您在企業中善用 AWS CloudFormation StackSets。本節列出的範本可支援 AWS CloudTrail 或 AWS Config 及其所包含的規則。

重要

作為允許 AWS Config 存取 Amazon S3 儲存貯體時的安全最佳實務，強烈建議您使用 AWS:SourceAccount 條件來限制儲存貯體政策中的存取。新範本會更新為具有 AWS:SourceAccount。如果您現有的儲存貯體政策未遵循此安全最佳作法，強烈建議您編輯該儲存貯體政策以納入此保護。這可確保 AWS Config 僅代表預期的使用者授予存取權。
