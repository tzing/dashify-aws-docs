AWS::CloudFormation::Interface ParameterLabel

ParameterLabel 是 AWS::CloudFormation::Interface 資源的屬性，指定易用的參數名稱或描述；AWS CloudFormation 主控台會顯示這些名稱或描述，而非參數的邏輯 ID。

語法

JSON

{
  "ParameterLogicalID" : Label
}

YAML

ParameterLogicalID: Label

屬性

ParameterLogicalID

參數的標籤。此標籤會定義建立或更新堆疊時， CloudFormation 主控台會顯示在 [指定參數] 頁面上的易記名稱或說明。ParameterLogicalID 金鑰必須是有效參數的邏輯 ID (區分大小寫)，而且這些參數必須已在範本的 Parameters 區段中宣告。

必要：否

Type (類型)：AWS::CloudFormation::Interface Label

文件慣用形式
ParameterGroup
