CloudFormation 資源提供者綱要

除了資源規格之外， CloudFormation還可以依區域提供 Amazon 資源的資源提供者結構描述。每個資源提供者結構描述都是 JSON 格式的文字檔，用來定義特定資源的內容和屬性。

針對每個區域， CloudFormation 提供一個 zip 檔案，其中包含該區域中每個可用資源的個別結構描述檔案。若要下載特定區域的資源提供者結構描述，請參閱下表。您也可以使用 CloudFormation 登錄檔來檢視資源的提供者結構描述。

資源可用性會因區域而異。若要查看指定區域中的資源可用性，請參閱該區域的資源提供者結構描述。

若要進一步瞭解資源提供者結構描述 (包括結構描述格式和定義)，請參閱《CloudFormation 命令列介面使用者指南》中的〈資源提供者
