取消堆疊更新

堆疊更新開始之後，如有堆疊仍在 UPDATE_IN_PROGRESS 狀態，您可以取消堆疊更新。更新完成之後，您便無法將其取消。不過，您可以使用任何之前的設定重新更新堆疊。

如果您取消堆疊更新，堆疊會復原至啟動堆疊更新之前存在的堆疊組態。

主題
  • 使用主控台取消堆疊更新
  • 使用命令列取消堆疊更新

使用主控台取消堆疊更新

  1. 從主 AWS CloudFormation 控台的堆疊清單中，選取目前正在更新的堆疊。其狀態必須為 UPDATE_IN_PROGRESS (正在更新)。

  2. 選擇 Stack actions (堆疊動作)，然後 Cancel update stack (取消更新堆疊)。

  3. 若要繼續取消更新，請選擇 Cancel update (取消更新)。否則，請選擇 Cancel (取消) 繼續更新。

堆疊會進入 UPDATE_ROLLBACK_IN_PROGRESS 狀態。更新取消完成之後，堆疊會設為 UPDATE_ROLLBACK_COMPLETE。

使用命令列取消堆疊更新

  • 使用命令 aws cloudformation cancel-update-stack 以取消更新。
