在 AWS CloudFormation 主控台上建立堆疊

在您建立堆疊之前，您必須有一個範本說明 AWS CloudFormation 會在您堆疊中包含的資源。如需更多詳細資訊，請參閱 使用 AWS CloudFormation 範本。

注意

您可使用變更集預覽新堆疊的組態。

在 CloudFormation 主控台上建立堆疊是一個簡單的精靈驅動程序，包含以下步驟：

  1. 啟動 Create Stack (建立堆疊) 精靈

  2. 選取堆疊範本

  3. 指定堆疊參數

  4. 設定 AWS CloudFormation 堆疊選項

  5. 檢閱您的堆疊

建立堆疊後，您可以監控堆疊的進度、檢視堆疊的資源和輸出、更新堆疊，以及刪除堆疊。如需這些動作的資訊，請參閱其相關主題。

啟動 Create Stack (建立堆疊) 精靈

在 CloudFormation 主控台上建立堆疊

  1. 在以下網址開啟 AWS CloudFormation 主控台：https://console.aws.amazon.com/cloudformation。

  2. 使用以下選項之一建立新的堆疊：

    • 選擇 Create Stack (建立堆疊)。如果您目前有堆疊正在執行，這是「唯一的」選項。

    • 在 Stacks (堆疊) 頁面上選擇 Create Stack (建立堆疊)。只有當無堆疊執行時才看得到此選項。

接著，選擇堆疊範本。

登入主控台
選取堆疊範本
