適用於 Microsoft Windows 的 Amazon Machine Image (AMI) 與 AWS CloudFormation 範本

您可以透過 AWS CloudFormation 來建立 Microsoft Windows 堆疊，進而執行 Windows 伺服器執行個體。在 AWS CloudFormation 範例範本頁面上，有一些預先設定的範本可直接啟動，例如下列範本：

  • Windows_Single_Server_SharePoint_Foundation.template – 在 Microsoft Windows Server® 2008 R2 上執行的 SharePoint® Foundation 2010

  • Windows_Single_Server_Active_Directory.template – 建立在 Microsoft Windows Server® 2008 R2 上執行的單一伺服器安裝。

  • Windows_Roles_And_Features.template – 建立單一伺服器，並指定 Microsoft Windows Server® 2008 R2 上執行的伺服器角色。

  • ElasticBeanstalk_Windows_Sample.template – 在執行 IIS 7.5 的 Windows Server 2008 R2 上啟動 AWS Elastic Beanstalk 範例應用程式。

注意

Microsoft、Windows Server 與 SharePoint 是 Microsoft 公司集團的商標。

儘管系統已預先設定這些堆疊，您仍可以採用任何 EC2 Windows AMI 作為 AWS CloudFormation Windows 堆疊的基礎。
