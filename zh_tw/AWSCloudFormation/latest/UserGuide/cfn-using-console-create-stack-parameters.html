指定堆疊名稱和參數

在選取堆疊範本之後，指定在範本中定義的堆疊名稱和參數值。

您可以使用參數在建立時自訂您的堆疊。您的參數值可用於堆疊範本，以修改資源的設定方式。使用此方式，您無需在多個範本中使用硬式編碼值，也能指定不同的設定。如需有關 AWS CloudFormation 範本中參數的詳細資訊，請參閱參數。

指定堆疊名稱和參數值

  1. 在 Specify stack details (指定堆疊詳細資訊) 頁面的 Stack name (堆疊名稱) 方塊中，輸入堆疊名稱。

    堆疊名稱是一個識別碼，可協助您從堆疊清單中找到特定堆疊。堆疊名稱只能包含英數字元 (區分大小寫) 和連字號。必須以字母字元開頭，且長度不可超過 128 個字元。

  2. 在 Parameters (參數) 部分中，指定堆疊範本中定義的參數。

    您可以用預設值來使用或變更任何參數。

  3. 當您滿意參數值時，請選擇 Next (下一步) 以繼續設定堆疊的選項。

AWS 特定參數類型

您建立包含 AWS 特定參數類型的堆疊時，CloudFormation 主控台會提供參數有效值的下拉式清單。根據參數類型，您可以依據 ID、名稱或 Name 標籤值來搜尋值。例如，您可以使用 AWS::EC2::VPC::Id 參數搜尋特定 VPC ID，例如 vpc-b47658d1。如果 VPC 已標記名稱，例如 Name:TestVPC，您也可以搜尋 TestVPC。目前，您只能以 Name 金鑰搜尋標籤值。

注意

主控台不提供下拉式清單或您無法搜尋 AWS::EC2::Image::Id 參數類型的值。CloudFormation 只會驗證輸入值是否為有效的 Amazon Elastic Compute Cloud 映像 ID。

群組和排序參數

主控台依照字母順序列出其邏輯 ID 的輸入參數。當您建立範本時，可使用 AWS::CloudFormation::Interface 中繼資料金鑰來覆寫預設的排序。如需詳細資訊與 AWS::CloudFormation::Interface 中繼資料金鑰的範例，請參閱 AWS::CloudFormation::Interface。
