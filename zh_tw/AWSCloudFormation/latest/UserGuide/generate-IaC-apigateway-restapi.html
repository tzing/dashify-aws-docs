更新AWS::ApiGateway::RestAPI資源

當產生的範本包含AWS::ApiGateway::RestApi資源時，會產生警告Body，指出BodyS3Location、和CloneFrom屬性會識別為UNSUPPORTED_PROPERTIES。這是因為這些是選擇性的唯寫屬性。IaC 產生器不知道這些屬性是否曾經套用至資源。IaC 產生器不會在產生的範本中包含這些屬性。

如果要為 REST API 設置Body屬性，可以使用以下步驟更新生成的模板：

  1. 使用 Amazon API Gateway GetExportAPI 動作來下載 API。例如，使用指aws apigateway get-exportAWS CLI令。如需詳細資訊，請參閱 API Gateway 開發人員指南中的從 API Gateway 匯出 REST API。

  2. 從 GetExport API 動作的回應中擷取Body屬性。將其上傳到 Amazon S3 存儲桶。

  3. 下載產生的範本。

  4. 將BodyS3Location/Bucket和BodyS3Location/Key屬性新增至範本，指定儲存貯體名稱和金鑰。Body

  5. 在 IaC 生成器控制台中打開生成的模板，然後選擇導入編輯的模板。
