AWS CloudFormation 入門

AWS CloudFormation 可以啟動許多不同類型的資源，因此本入門演練僅會論及幾個簡單的概念，協助您了解 AWS CloudFormation 的使用方式。

在本節中，您會使用 AWS Management Console 在 AWS CloudFormation 範例範本程式庫提供的範例範本中建立堆疊，並了解建立範本的基本概念。

在以下演練中，我們會使用簡易的範本來啟動、更新與刪除堆疊。掌握基本概念後，即可進一步了解如何建立較複雜的範本和堆疊。

AWS CloudFormation 讓部署一組 Amazon Web Services (AWS) 資源就像提交範本一樣簡單。範本是一種用來說明堆疊的文字檔案，而堆疊則是要以群組方式共同部署的一系列 AWS 資源。您可以透過範本來定義要存放在堆疊中的所有 AWS 資源，包括 Amazon Elastic Compute Cloud 執行個體、Amazon Relational Database Service 資料庫執行個體，以及其他資源。如需資源類型的清單，請參閱 AWS 資源和屬性類型參考。
