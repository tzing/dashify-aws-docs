自動縮放 CloudFormation 範本片段

使用 Amazon EC2 自 Auto Scaling，您可以使用擴展政策或排程擴展來自動擴展 Amazon EC2 執行個體。Auto Scaling 群組是 Amazon EC2 執行個體的集合，可啟用自動擴展和叢集管理功能，例如擴展政策、排程動作、運作狀態檢查、生命週期勾點和負載平衡。

Application Auto Scaling 可透過擴展政策或排程擴展，提供 Amazon EC2 以外的不同資源的自動擴展。

您可以使用AWS CloudFormation範本建立和設定 Auto Scaling 群組、擴展政策、排程動作和其他自動擴展資源，做為基礎結構的一部分。範本可讓您以可重複且一致的方式輕鬆管理和自動化自動擴展資源的部署。

下列範例範本片段描述 Amazon EC2 自動擴展和 Application Auto Scaling 自動擴展的AWS CloudFormation資源或元件。這些程式碼片段設計用於整合至範本中，而且並非設計為獨立執行。
