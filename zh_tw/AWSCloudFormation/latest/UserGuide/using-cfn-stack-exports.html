匯出堆疊輸出值

若要在堆疊之間共享資訊，請匯出堆疊的輸出值。同一個 AWS 帳戶和區域中的其他堆疊可以匯入這些匯出的值。例如，您可以讓單一聯網堆疊匯出公有 Web 伺服器的子網路和安全群組 ID。含公有 Web 伺服器的堆疊即可輕鬆匯入這些聯網資源。您不必在堆疊範本中硬式編碼資源 ID，也不需要以輸入參數形式傳遞 ID。

若要匯出堆疊的輸出值，請使用堆疊範本中輸出區段的 Export 欄位。若要匯入這些值，請在其他堆疊的範本中使用 Fn::ImportValue 函數。如需演練和範例範本，請參閱演練：參考在另一個 AWS CloudFormation 堆疊中的資源輸出。

注意

當其他堆疊匯入輸出值之後，您就無法刪除匯出輸出值的堆疊，也無法修改匯出的輸出值。您必須先移除所有匯入，之後才可以刪除匯出堆疊或修改輸出值。

匯出堆疊輸出值與使用巢狀堆疊的比較

巢狀堆疊是指藉由使用 AWS::CloudFormation::Stack 資源在另一個堆疊中建立的堆疊。使用巢狀堆疊時，您可以透過單一堆疊來部署和管理所有資源。您可以將巢狀堆疊群組中某個堆疊的輸出，作為群組中其他堆疊的輸入。這與匯出值的情況不同。

如果您想要將資訊的共享隔離在巢狀堆疊群組內，則建議您使用巢狀堆疊。若要與其他堆疊 (而不只是在巢狀堆疊的群組內) 共享資訊，則匯出值。例如，您可以建立含子網路的單一堆疊，然後匯出其 ID。其他堆疊只要匯入該子網路的 ID 即可使用，且每個堆疊不需要建立自己的子網路。只要有堆疊匯入子網路 ID，您就無法變更或刪除該 ID。

列出已匯出的輸出值

若要查看您可以匯入哪些值，請使用 AWS CloudFormation 主控台、AWS CLI 或 AWS CloudFormation API，列出所有已匯出的輸出值。AWS CloudFormation 會針對目前匯出輸出的來源區域和堆疊，顯示已匯出的輸出名稱和值。若要參考堆疊範本中的已匯出輸出值，請使用匯出名稱和 Fn::ImportValue 函數。

列出已匯出的輸出值 (主控台)

  • 在 AWS CloudFormation 主控台中，從 CloudFormation 導覽窗格選擇 Exports (匯出)。

列出已匯出的輸出值 (AWS CLI)

  • 執行 aws cloudformation list-exports 命令。

列出已匯出的輸出值 (API)

  • 執行 ListExportsAPI 作業。
