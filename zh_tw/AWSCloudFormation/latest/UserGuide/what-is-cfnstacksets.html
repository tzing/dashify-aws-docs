使用 AWS CloudFormation StackSets

AWS CloudFormation StackSets 透過讓您透過單一作業跨多個帳戶建立、更新或刪除堆疊， AWS 區域 藉此擴充堆疊的功能。您可以使用管理員帳戶定義並管理 AWS CloudFormation 範本，並使用該範本做為將堆疊佈建到指定目標帳戶中的基礎 AWS 區域。

如需有關 StackSets 區域支援的資訊，請參閱區StackSets 域支援。

本節可協助您開始使用 StackSets，並回答有關如何使用堆疊集建立、更新和刪除的常見問題，以及疑難排解。
