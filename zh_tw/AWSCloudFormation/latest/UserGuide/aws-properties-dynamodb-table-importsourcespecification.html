AWS::DynamoDB::Table ImportSourceSpecification

指定從 S3 儲存貯體來源匯入資料表的資料屬性。

語法

若要在您的 AWS CloudFormation 範本中宣告此實體，請使用下列語法：

JSON

{
  "InputCompressionType" : String,
  "InputFormat" : String,
  "InputFormatOptions" : InputFormatOptions,
  "S3BucketSource" : S3BucketSource
}

YAML

  InputCompressionType: String
  InputFormat: String
  InputFormatOptions: 
    InputFormatOptions
  S3BucketSource: 
    S3BucketSource

屬性

InputCompressionType

要對來自已匯入表格輸入使用的壓縮類型。

必要：否

類型：字串

允許值：GZIP | ZSTD | NONE

更新要求：替換

InputFormat

來源資料的格式。ImportFormat 的有效值為 CSV、DYNAMODB_JSON 或 ION。

必要：是

類型：字串

允許值：DYNAMODB_JSON | ION | CSV

更新要求：替換

InputFormatOptions

指定輸入格式化方式的其他屬性，

必要：否

類型:InputFormatOptions

更新要求：替換

S3BucketSource

提供匯入來源的 S3 儲存貯體。

必要：是

類型：中三 BucketSource

更新要求：替換
