在 AWS CloudFormation 主控台中刪除堆疊

刪除 堆疊

  1. 在以下網址開啟 AWS CloudFormation 主控台：https://console.aws.amazon.com/cloudformation。

  2. 在 CloudFormation 主控台的 Stacks (堆疊) 頁面上，選取您要刪除的堆疊。此堆疊目前必須正在執行。

  3. 在 stack details (堆疊詳細資訊) 窗格中，選擇 Delete (刪除)。

  4. 當系統提示時，選取 Delete stack (刪除堆疊)。

    注意

    堆疊刪除操作開始後將無法停止。堆疊繼續進行到 DELETE_IN_PROGRESS (正在刪除) 狀態。

    在堆疊刪除完成之後，堆疊即處於 DELETE_COMPLETE (刪除完成) 狀態。根據預設，CloudFormation 主控台不會顯示狀態為 DELETE_COMPLETE (刪除完成) 的堆疊。若要顯示已刪除的堆疊，您必須按照在 AWS CloudFormation 主控台上檢視已刪除的堆疊所述變更堆疊檢視的篩選條件。

    如果刪除失敗，堆疊會處於 DELETE_FAILED (刪除失敗) 狀態。如需問題的解決方法，請參閱刪除堆疊失敗的故障診斷主題。

如需如何防止堆疊遭意外刪除的資訊，請參閱防止堆疊遭刪除。
