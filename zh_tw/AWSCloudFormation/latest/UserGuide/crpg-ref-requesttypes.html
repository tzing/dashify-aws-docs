自訂資源請求類型

範本開發人員建立、更新或刪除包含自訂資源的堆疊時，請求類型會在 RequestType 傳送之廠商請求物件中的 AWS CloudFormation 欄位內傳送。

每種請求類型都具備與請求一同傳送的特定欄位集，包含自訂資源提供者回應的 Amazon S3 URL。提供者必須在一小時內以 SUCCESS 或 FAILED 結果回應 S3 儲存貯體。在一小時之後，請求便會逾時。每一個結果也都具有 AWS CloudFormation 預期的特定欄位組。

本節提供請求每一種請求類型的請求和回應欄位相關資訊，以及範例。
