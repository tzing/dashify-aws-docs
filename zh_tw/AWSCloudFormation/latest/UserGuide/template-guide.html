使用 AWS CloudFormation 範本

您必須了解 AWS CloudFormation 範本 (這些範本屬於 JSON 或 YAML 格式的文字檔)，才能佈建並設定堆疊資源。而您亦可以透過這些範本的說明，了解欲在 AWS CloudFormation 堆疊中佈建的資源。您可以使用 AWS CloudFormation 設計工具或任何文字編輯器來建立和儲存範本。如需範本結構與語法的資訊，請參閱 範本剖析。

如果您不熟悉 JSON 或 YAML，您可以使用 AWS CloudFormation 設計工具協助您開始使用 AWS CloudFormation 範本。AWS CloudFormation設計工具是一種工具，能讓使用者以視覺化的方式建立與修改範本。如需詳細資訊，請參閱 什麼是 AWS CloudFormation 設計工具？。

範本程式碼片段提供範例，向您示範如何撰寫特定資源的範本。舉例來說，您可以檢視 Amazon EC2 執行個體、Amazon S3 網域、AWS CloudFormation 映射等項目的程式碼片段。程式碼片段依資源分組，並搭配一般範本程式碼片段中的一般用途 AWS CloudFormation 程式碼片段。

如需範本中可使用的受支援資源、類型名稱、內部函數與虛擬參數的詳細資訊，請參閱範本參考。
